Use input controls to facilitate user input inside your app. The input controls comprise of the following:

* [Button](https://docs.uipath.com/apps/automation-suite/latest/user-guide/button#button)
* [Checkbox](https://docs.uipath.com/apps/automation-suite/latest/user-guide/checkbox#checkbox)
* [Date Picker](https://docs.uipath.com/apps/automation-suite/latest/user-guide/date-picker#date-picker)
* [Dropdown](https://docs.uipath.com/apps/automation-suite/latest/user-guide/dropdown#dropdown)
* [File Uploader](https://docs.uipath.com/apps/automation-suite/latest/user-guide/file-uploader#file-uploader)
* [Multiselect dropdown](https://docs.uipath.com/apps/automation-suite/latest/user-guide/multiselect-dropdown#multiselect-dropdown)
* [Radio Button](https://docs.uipath.com/apps/automation-suite/latest/user-guide/radio-button#radio-button)
* [Rich Text Editor](https://docs.uipath.com/apps/automation-suite/latest/user-guide/rich-text-editor#rich-text-editor)
* [Slider](https://docs.uipath.com/apps/automation-suite/latest/user-guide/slider#slider)
* [Switch](https://docs.uipath.com/apps/automation-suite/latest/user-guide/switch#switch)
* [Text Area](https://docs.uipath.com/apps/automation-suite/latest/user-guide/text-area#text-area)
* [Textbox](https://docs.uipath.com/apps/automation-suite/latest/user-guide/textbox#textbox)
* [Textbox (Email)](https://docs.uipath.com/apps/automation-suite/latest/user-guide/textbox-email#textbox-(email))
* [Textbox (Number)](https://docs.uipath.com/apps/automation-suite/latest/user-guide/textbox-number#textbox-(number))
