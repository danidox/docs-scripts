The **Importing** option allows you to create a new app from a file.

:::note
Apps created using Automation Suite Apps cannot be imported in Automation Cloud<sup>TM</sup> instances.
:::

## Creating a New App From a File

1. On the **Build** tab, click the **Create new app** button.
2. Give the new app a name.
3. Click on the **Import from file** link.
4. Browse to the `*.uiapp` file you wish to import and click Open.
5. Click the **Create** button.

   ![docs image](/images/apps/apps-docs-image-93711-3036a294.webp)

The new app will be created and you'll see the main page for the app.
