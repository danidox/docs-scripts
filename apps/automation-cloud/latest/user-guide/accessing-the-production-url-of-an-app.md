Once an app was deployed to an Orchestrator folder, you can access the production URL and use it to embed the app in other applications or web pages.

There are two ways in which you can access the production URL:

* On the **Apps &gt; Run** tab, select the deployed app, click **More options**, then select **Copy URL**.

  ![docs image](/images/apps/apps-docs-image-296740-bdf216a4.webp)
* On the **Orchestrator** &gt; **Automations** &gt; **[folder_name]** &gt; **Apps** tab, select the desired app, click **More options**, then select **Copy URL**.

  ![docs image](/images/apps/apps-docs-image-296744-eebd482b.webp)
