You can bring all your documents and media files inside the App Studio, for easy access and management, and to bind their properties to your app controls or rules.

The following formats are supported:

* `.bmp`
* `.png`
* `.jpeg` /`.jpg`
* `.gif`
* `.webp`
1. In your app, expand the **Add control** menu, then select **Media**. The file browser opens.

   ![docs image](/images/apps/apps-docs-image-453513-fbe45425.webp)
2. Search for the media file you want to add to your app, then click **Open**.
   :::note
   * Maximum allowed file size is 5 MB.
   * You can only add one file at a time.
   :::
3. On the left-side panel, the **Media** section is displayed. Expand it to see all the uploaded files.
