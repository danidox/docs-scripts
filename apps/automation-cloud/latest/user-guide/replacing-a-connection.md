Within **Apps**, you can replace any connection referenced in the app with another. This is helpful when moving apps between organizations or environments.

To replace a connection, follow these steps:

1. Go to the app where you want to replace a connection.
2. In the left-hand Hierarchy panel, right-click on the connection you want to replace. Select **Replace**.
3. Select the dropdown menu. A new option appears. This entry shows your current workspace.
4. Hover over the new option. To the right of the dropdown menu, a list of options is now displayed. The current connection is always the first option.
5. If other connections are available, select the one you need.
6. Select **Replace**.
   :::note
   If your connection is broken, you may receive an email from Integration Service. In this case, you need to replace the broken connection with a working one.
   :::

The new connection is available for use in Apps.
