Apps in Studio Web represent a new integration between Apps and Studio Web.

This integration enables you to build, test, and deploy apps with inline automation, executed by serverless robots.

This enables faster development within a single environment, minimizing context switching and facilitating uninterrupted interactions between your automation and your users.

For more details, refer to the [app projects documentation in Studio Web](https://docs.uipath.com/studio-web/automation-suite/2.2510/user-guide/apps-in-studio-web).
