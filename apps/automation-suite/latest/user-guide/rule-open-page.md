Use the **Open a page** rule to open an Apps page as a pop-up or as the content of a page container.

![docs image](/images/apps/apps-docs-image-290854-ac6c7f65.webp)

## What Page to Open

Clicking the **What page to open** field opens the **Resources** panel, which displays the available Apps pages.

Select the page you want to open by double-clicking on it.

## Show as Pop-Over (Modal)

If true, displays the page as a modal dialog. **False** by default.

:::note
When a page is opened as a dialog, always provide the width of the page you want to open in pixels.
:::

## Show as Bottom Sheet

If true, opens the page in a small container at the bottom of the screen. **False** by default.

## Prevent Close on Click

If false, closes the page when you click outside of it. **False** by default.

## Open in Page Container (optional)

Clicking the **Open in Page Container (optional)** field opens the **Resources** panel, which displays the available Apps pages.

Select the page to host the page selected in the **What page to open** field by double-clicking on it.

:::note
This setting applies when the **Show as Pop-Over** setting is false.
:::
