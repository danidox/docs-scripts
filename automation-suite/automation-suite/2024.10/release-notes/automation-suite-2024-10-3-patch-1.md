**Release date: May 16, 2025**

## About this patch

This release applies a patch to Automation Suite 2024.10.3. The patch addresses some issues in Automation Hub. For more details, see the individual product release notes referenced in the [Product versions](https://docs.uipath.com/automation-suite/automation-suite/2024.10/release-notes/automation-suite-2024-10-3-patch-1#product-versions) section.

:::important
This fix will also be included in all future releases starting with Automation Suite 2024.10.4. We recommend that you wait for Automation Suite 2024.10.4 to get this fix.
:::

## How to apply the patch

To apply this patch, follow the instructions in [Applying a patch](https://docs.uipath.com/automation-suite/automation-suite/2024.10/installation-guide/applying-a-patch).
