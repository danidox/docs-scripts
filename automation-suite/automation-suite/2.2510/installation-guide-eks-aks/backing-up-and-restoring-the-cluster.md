Automation Suite provides backup and restore functionality to help prevent data loss in various scenarios. You can configure backups at any time after installation. Although backups are optional, enabling them is strongly recommended to ensure that you can recover the environment to a consistent state in the event of a disaster.

Backups use the cloud provider’s managed backup solution, which leverages native cloud capabilities to unify and simplify your organization’s backup and recovery strategy.
