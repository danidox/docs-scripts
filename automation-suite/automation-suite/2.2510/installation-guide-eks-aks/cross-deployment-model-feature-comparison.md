If you need to deploy Automation Suite in different supported environments, refer to the following guides:

* [Automation Suite on OpenShift installation guide](https://docs.uipath.com/automation-suite/automation-suite/2.2510/installation-guide-openshift/automation-suite-overview)
* [Automation Suite on Linux installation guide](https://docs.uipath.com/automation-suite/automation-suite/2.2510/installation-guide/automation-suite-overview)
