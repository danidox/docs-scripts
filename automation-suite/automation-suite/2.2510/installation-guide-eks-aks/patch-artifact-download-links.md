This page provides the download links for the artifacts required to apply a patch.

## 2.2510.0 patches

This section provides the download links for the artifacts required for Automation Suite on EKS/AKS 2.2510.0 patches.

### 2.2510.0+patch1

#### uipathctl

To download `uipathctl` for Automation Suite 2.2510.0+patch1, run one of the following commands, depending on your OS:

* **Linux**:
  ```
  wget -O ~/uipathctl-linux-amd64.tar.gz https://download.uipath.com/uipathctl/2.2510.0-patch1/uipathctl-2.2510.0-patch1-linux-amd64.tar.gz
  ```
* **Windows:**
  ```
  wget -O ~/uipathctl-windows-amd64.tar.gz https://download.uipath.com/uipathctl/2.2510.0-patch1/uipathctl-2.2510.0-patch1-windows-amd64.tar.gz
  ```
* **MacOS:**
  ```
  wget -O ~/uipathctl-darwin-amd64.tar.gz https://download.uipath.com/uipathctl/2.2510.0-patch1/uipathctl-2.2510.0-patch1-darwin-amd64.tar.gz
  ```
