**Release date: December 5, 2025**

## About this patch

This release applies a patch to Automation Suite 2.2510.0. The patch addresses some issues in Studio Web. For more details, see the individual product release notes referenced in the [Product versions](https://docs.uipath.com/automation-suite/automation-suite/2.2510/release-notes/automation-suite-on-openshift-2-2510-0-patch-1#product-versions) section.

:::important
This fix will also be included in all future releases starting with Automation Suite 2.2510.1 We recommend that you wait for Automation Suite 2.2510.1 to get this fix.
:::

## How to apply the patch

To apply this patch, follow the instructions in [Applying a patch](https://docs.uipath.com/automation-suite/automation-suite/2.2510/installation-guide-eks-aks/applying-a-patch).
