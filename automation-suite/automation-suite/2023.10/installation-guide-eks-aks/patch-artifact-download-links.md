This page provides the download links for the artifacts required to apply a patch.

## 2023.10.12 patches

This section provides the download links for the artifacts required for Automation Suite on EKS/AKS 2023.10.12 patches.

### 2023.10.12+patch1

#### as-cmk.tar.gz

To download `as-cmk.tar.gz` for Automation Suite 2023.10.12+patch1, run the following command:

```
wget -O ~/as-cmk.tar.gz https://download.uipath.com/automation-suite/2023.10.12+patch1/as-cmk-2023.10.12+patch1.tar.gz
```
