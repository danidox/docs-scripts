To start the migration operation, take the following steps:

1. Move the Identity organization data from standalone to Automation Suite. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/moving-the-identity-organization-data-from-standalone-to-automation-suite#step-1%3A-moving-the-identity-organization-data-from-standalone-to-automation-suite)
2. Restore the standalone product database. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/restoring-the-standalone-product-database#step-2%3A-restoring-the-standalone-product-database)
3. Back up the platform database in Automation Suite. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/backing-up-the-platform-database-in-automation-suite#step-3%3A-backing-up-the-platform-database-in-automation-suite)
4. Merge the organizations in Automation Suite. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/merging-organizations-in-automation-suite#step-4%3A-merging-organizations-in-automation-suite)
5. Update the migrated product connection strings. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/updating-the-connection-strings#step-5%3A-updating-the-migrated-product-connection-strings)
6. Migrate standalone Orchestrator. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/migrating-standalone-orchestrator#step-6%3A-migrating-standalone-orchestrator)
7. Migrate standalone Insights. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/migrating-standalone-insights#step-7%3A-migrating-standalone-insights)
8. Delete the default tenant. [Details...](https://docs.uipath.com/automation-suite/automation-suite/2023.10/installation-guide-eks-aks/deleting-the-default-tenant#step-9%3A-deleting-the-default-tenant)
