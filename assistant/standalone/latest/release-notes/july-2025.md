## July 31, 2025

**Build number: 2025.0.172**

### Organization switcher

You can now switch between your recently accessed organizations using the **Organization** dropdown in the Assistant **Preferences** menu. The Assistant uses cached sessions to log you in automatically, saving time and effort. If a session has expired, you’ll be guided through the usual logout and login process.

### Shorcut to Studio Web templates

You can access your Studio Web templates directly from the Search bar by typing the name of template you need.

Alternatively, you can view the entire collection by navigating to the Templates page in Studio Web using the new shortcut available in Assistant. This shortcut is located under the **UiPath Products** section within the Dashboard panel.
