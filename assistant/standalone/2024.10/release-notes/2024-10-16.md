**Release date: September 3, 2025**

## Bug fixes

* Fixed a bug that caused the Preferences screen to display a blank page instead of the correct display.
* Resolved issues with running applications from Apps in proxy-enabled environments. A new Automation Ops policy setting ensures a smooth operation across different network configurations. This setting allows you to specify URLs that should bypass the proxy client, offering greater flexibility and control over network access.
* Updated Electron dependencies to enhance security and prevent potential vulnerabilities.
