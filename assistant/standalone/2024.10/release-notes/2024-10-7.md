**Release Date: December 4, 2024**

We are increasing our Assistant version to maintain alignment with Studio and Robot versions. This update does not include any fixes.

## New documentation structure

Starting with this version, the Assistant documentation separates from the UiPath Robot guide and provides dedicated user and release notes guides.

Read the [UiPath Assistant user guide](https://docs.uipath.com/assistant/standalone/2024.10/user-guide/about-uipath-assistant).
