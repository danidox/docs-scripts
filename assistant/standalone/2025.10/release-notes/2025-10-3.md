**Release date: December 11, 2025**

## Bug fixes

* We fixed an issue where enabling a **Troubleshooting Session** for an unattended session from Orchestrator caused UiPath® Assistant to behave incorrectly.
* Previously, the **Read the docs** hyperlink in **Preferences → Help & Info → Read the docs** pointed to the wrong documentation page.
* Addressed an issue where the **Run in PiP** setting in Assistant was not applied when starting a process.
