**Release Date: 13 June 2023**

The [UiPath Assistant for Excel add-in](https://docs-staging.uipath.com/robot/standalone/2023.10/user-guide/excel-add-in) is now generally available, and can be downloaded from the official Microsoft Store, or manually configured using the manifest file.

The add-in allows you to use UiPath Assistant capabilities in the native Excel application. This provides an easy way to run, manage, and configure Excel-specific automations without having to switch between applications.
