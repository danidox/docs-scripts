**Release date: August 7, 2024**

## Improvements

For accessibility reasons, screen readers mention the name of the automation when you select the **Run** button in UiPath Assistant.

## Bug fixes

* An executor issue caused the `Exception.Data` JSON to incorrectly display the values of the following properties:
  + ActivityFullName
  + ActivityId
  + DisplayName
  + WorkflowFileThis occurred while debugging projects in Studio.
