You can customize the following options for Assistant Web policies:

## Widgets

Widgets are plugins that extend the Assistant Web capabilities, such as allowing access to different products directly from Assistant Web.


:::important
To use widgets that allow integration between Assistant and other UiPath products, make sure that those products are available on your tenant.
:::

The following widgets are enabled by default:

* Apps
* Autopilot

## Feature toggles

Feature toggles are settings that allow you to enable or disable specific functionalities for Assistant Web.

The following feature toggles are available for configuration:

* **Enable Task Capture—**Select **Yes** to show the Task Capture Launcher.
* **Group processes by folders—**Select **Yes** to group processes by Orchestrator folder on the home page. To allow users to change this option from Assistant Web, select **Use local user preferences**.
* **Enable Action Center—**Select **Yes** to allow the use of the Action Center widget.
* **Automatically switch to Jobs Tab when an automation starts—**Select **Yes** to make Assistant Web focus on the **Jobs** tab when an automation execution starts. To allow users to change this option from Assistant Web, select **Use local user preferences**.
* **Enable Studio Web—**Select **Yes** to allow users to launch Studio Web from the UiPath Products section in Assistant Web and return Studio Web templates in Assistant search view.
* **Show Install Assistant button—**Select **Yes** to allow the display of the Install Assistant button.

Select **Save** to confirm your changes.
