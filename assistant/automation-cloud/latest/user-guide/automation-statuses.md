title: "Automation statuses"
visible: true
slug: "automation-statuses"
---

After you start an automation, it can have different statuses:

* **Pending**—No machine is available. The automation starts when one becomes available.
* **Running—**The job is in progress.
* **Suspended**—The job is waiting for an action to complete.
* **Success—**The job completed successfully.
* **Failed**—The job did not start or complete.
