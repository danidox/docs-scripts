---
title: "Installing Assistant standalone"
visible: true
slug: "installing-assistant-standalone"
---

If you prefer to install Assistant on your device, you have the option to download the latest Assistant version and install it. To do that, select **Install Assistant** next to the **Preferences** button to start the download.


:::important
Once triggered, the download operation detects your operating system and the current architecture, and downloads the relevant installer: `UiPathRobot.msi` for Windows OS, and DMG file for Intel Macs - x64 or Apple Silicon OS.
:::
