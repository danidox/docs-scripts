Assistant Web enables you to:

* Find all the automations you are authorized to run, determined by your roles and folder permissions.
* Configure connections and input arguments for the automations.
* Run and monitor foreground or background unattended automations, with an unattended setup in place.
* Add frequently used automations to a Favorites section.
* Launch Apps, Action Center, Task Mining, or Studio Web.
* Execute background automations with only an attended robot.
