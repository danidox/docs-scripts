You can create and customize policies for Assistant and Assistant Web from [Automation Ops > Governance](https://docs.uipath.com/automation-ops/automation-cloud/latest/user-guide/configure-a-governance-policy).

These policies help you control how Assistant behaves in the tenant where they are deployed, or they can be applied to specific users, so you can enforce a behavior for the selected user.
