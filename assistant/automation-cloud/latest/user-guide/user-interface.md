The user interface of Assistant in Automation Cloud™ resembles the user interface of on-premises Assistant. There are two main panels:

* The left-hand panel contains the **Automations** panel.
* The right-hand panel also referred to as "the Launchpad", displays your favorite automations, custom automation groups, and UiPath products for quick launch. Additionally, it displays details of the automation selected in the left-hand panel.
