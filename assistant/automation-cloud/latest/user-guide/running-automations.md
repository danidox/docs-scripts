title: "Running automations"
visible: true
slug: "running-automations"
---

To run an automation, you have two options:

* In the **Automations** list, select **Run** for the desired automation.
* Select an automation, then in the right-side panel, go to the **Details** tab and select **Run** at the bottom left.

If the automation requires configuration, the **Configure** button appears instead. After you provide the details, the button changes to **Run**.
