If Autopilot for everyone is installed in the tenant where you are currently signed in, the Autopilot tab is displayed.

Select it to initiate the chat experience with Autopilot, and [learn how you can interact with it](https://docs.uipath.com/autopilot/automation-cloud/latest/user-guide/prerequisites-to-use-autopilot-in-assistant).
