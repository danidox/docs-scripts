Preferences window allows you to custom your Assistant Web experience. It includes:

* **General preferences**—Enable or disable features such as grouping automations by folders, or focusing on the **Jobs** tab after an automation starts.
* **UiPath Products**—Control which UiPath products appear on the launchpad. The available option are manageable through Automation Ops policies.
* **Help & Info**—View details about the current **Assistant** cloud service, access help resources, or provide feedback.
* **Autopilot for Eveyone**—Configure settings specific to Autopilot for Everyone.
