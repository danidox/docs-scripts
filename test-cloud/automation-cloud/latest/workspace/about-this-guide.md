The Test Cloud documentation consolidates administrative and API information for all UiPath Test Cloud offerings. While most content applies across offerings, some features and capabilities vary depending on the cloud offering you use.

To help you understand how the information applies to your environment, this guide uses shared terminology and includes references to feature availability where differences exist.

## Cloud offering

**Cloud offering** refers to the UiPath cloud platform and its associated products and capabilities. The supported cloud offerings are:
* Test Cloud
* Test Cloud Public Sector
* Test Cloud Dedicated

The term cloud offering is used throughout the documentation when instructions, navigation paths, or behavior depend on the specific UiPath cloud platform you are using.

When the documentation refers to **your cloud offering**, follow the instructions in the Test Cloud instance that applies to your organization.

## Platform

The term **platform** refers to the Test Cloud platform generically, regardless of whether you use Test Cloud, Test Cloud Public Sector, or Test Cloud Dedicated. **Unless otherwise specified, the term applies equally to all cloud offerings.**

The platform includes shared administrative features and capabilities within a cloud offering, such as:

* Authentication and security features
* Organization- and tenant-level settings
* Administrative configuration available in the **Admin** section
* Homepage and organizational preferences

In these contexts, **platform** refers to the administrative environment of the Test Cloud platform, rather than to an individual offering.

## How to navigate the Test Cloud documentation

This guide contains shared content for all supported cloud offerings. Differences are indicated where applicable.

When reading the [Admin](https://docs.uipath.com/test-cloud/automation-cloud/latest/workspace/about-test-cloud#about-test-cloud) or [API](https://docs.uipath.com/automation-cloud/automation-cloud/latest/api-guide/about-this-guide) guides:

1. Follow the general instructions provided in the guide.
2. When you see a note stating that feature availability depends on the cloud offering, go to the [feature availability page](https://docs.uipath.com/test-cloud/automation-cloud/latest/workspace/test-cloud-feature-availability#test-cloud-product-and-feature-availability) page.
3. Review the information for the cloud offering you are using to confirm whether the feature is available and how it applies to your environment.
