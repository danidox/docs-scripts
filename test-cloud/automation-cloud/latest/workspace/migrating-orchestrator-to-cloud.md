You can migrate from standalone Orchestrator to a cloud offering, either [using the Migration Tool](https://docs.uipath.com/test-cloud/automation-cloud/latest/workspace/using-the-migration-tool#using-the-automation-cloud-migration-tool) or manually ([Manual Migration](https://docs.uipath.com/test-cloud/automation-cloud/latest/workspace/manual-migration#manual-migration)).

## Orchestrator Manager

Orchestrator Manager is a tool that helps Orchestrator administrators perform bulk operations on entities such as assets, environments, machines, processes, users, robots, organization units, folders, queues and packages.

For more information, visit the [Orchestrator Manager](https://marketplace.uipath.com/listings/orchestrator-manager) page.
