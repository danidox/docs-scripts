## Overview

UiPath® business extentions are applications that are external to your UiPath platform and can be granted secure access to UiPath resources without having to share user credentials.
:::note
UiPath business extentions are made available for consumption across all organizations by going through a rigorous validation process and then registered by an organization administrator.
:::

## Validation process

To make a UiPath business extention available and ready for registration, it must undergo a rigorous validation process. Once vetted, the application will show up in **Admin** > **External applications** page > **App registrations** tab > **Available applications**.

## Managing app registrations

### Registering an app

You have to be an Org Admin to perform this operation.

To register an app, follow these steps:

1. Go to **Admin** and select the organization at the top of the panel on the left.
2. Select **External applications**.
3. Select the **App registrations** tab to view the Registered and Available applications.

   !['Registering an app' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/325029)
4. Select on the contextual **Show more options** menu of an app in the **Available applications** section to register an app.
5. Select **Register**.

   !['Registering an app' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326064)

The "Application registered successfully." message is displayed to confirm that the app has been successfully registered and is ready for consumption in the organization.

   !['Application registered successfully' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326056)

### Unregistering an app

You have to be an Org Admin to perform this operation.

To register an app, follow these steps:

1. Go to **Admin** and select the organization at the top of the panel on the left.
2. Select **External applications**.
3. Select the **App registrations** tab to view the Registered and Available applications.

   !['Unregistering an app' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/325029)
4. Select on the contextual **Show more options** menu of an app in the **Registered applications section** to unregister an app.
5. Select **Unregister**.

   !['Unregistering an app' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326027)
6. A warning message will appear to confirm whether or not you want to continue unregistering the app. Select **Unregister**.

   !['Unregistering an app' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326033)

The "Application unregistered." message is displayed to confirm that the app has been successfully unregistered.

   !['Application unregistered' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326040)

### Viewing access scopes

To view the access scopes of registered and unregistered apps:

1. Go to **Admin** and select the organization at the top of the panel on the left.
2. Select **External applications**.
3. Select the **App registrations** tab to view the Registered and Available applications.

   !['Registered and Available applications' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/325029)
4. Select on the contextual **Show more options** menu of an app.
5. Select **View Details**.

   !['View details' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326110)
6. Select the **Details** tab on the leftside panel to view the access scopes defined for that app.

   !['Access scopes' image](https://docs-dev.uipath.com/api/binary/test-cloud/2/636657/326119)
