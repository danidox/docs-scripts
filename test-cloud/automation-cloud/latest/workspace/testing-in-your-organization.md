Test Cloud supports testing across the following major categories:

* **Automation testing**: It is also referred to as RPA testing, and focuses on validating your automations to ensure that the workflows you build in UiPath behave as expected.
* **Application testing**: Covers end-to-end testing of your enterprise applications, such as web or mobile apps, ensuring they meet your business and performance requirements. With the UiPath testing capabilities you can validate your automations and test external enterprise applications, regardless of whether they are automated using UiPath.
* Regardless of your use case, UiPath provides a unified ecosystem of tools that support the entire testing lifecycle: from test design and automation to execution, traceability, and AI-powered analysis.

## Automation testing

Automation testing helps ensure the reliability of your RPA workflows. UiPath helps you overcome common challenges such as complex process coverage, test data management, integration with ALM tools, and visibility over results. With UiPath, you can:

* Design low-code or coded tests in **Studio**.
* Execute tests through **Orchestrator**.
* Gain full artifact traceability and manage your test portfolio in **Test Manager**.
* Use Autopilot to generate test cases, evaluate requirements, and analyze test results using AI.

You can also perform data-driven testing, integrate with CI/CD pipelines, and measure test quality using coverage and performance profiling.

## Application testing

Application testing allows you to test enterprise apps with powerful UI and API automation. When performing application testing, Test Cloud offers the following:

* Flexible automation development in Studio STS, 2025.10 or Studio Web, using either low-code or full-code options.
* Advanced testing activities via the Testing and Mobile Automation activity packages.
* Scalable test execution using App Testing robots.
* Centralized test management and traceability with Test Manager.
* AI-enhanced workflows using Autopilot.
* Deep integration with ALM tools via Test Manager Connect or native connectors.
