If you are moving from the Flex licensing plan to Unified Pricing, you can use the License Migration wizard to complete the migration.

To migrate, take the following steps:

1. Log in to Automation Cloud™ as an organization administrator.
2. Go to **Admin** > **Licenses**, and select **License Migration** in the top-right corner of the page. The license migration view opens. Each Flex license eligible for migration is displayed as a distinct license card.
3. Select **Start Migration** for the license type that you want to migrate.
4. In the license migration view:
   1. Select the action that you want to perform. The dropdown menu displays all available Unified license SKUs that can be mapped to the selected Flex license, as well as the **Remove license** option.
   2. Review and select the targets:
      * **Users**: lists all individuals directly assigned to the Flex license.
      * **Groups**: which lists the groups that hold the Flex license.
5. Select **Apply Migration** to start the migration process.
:::note
For the recommended mappings of user licenses from the Flex plan to the Unified Pricing plan:
* select the **Migration Guide** icon in the lower -right corner of the migration page
* refer to the [Flex to Unified Pricing user license mapping](https://docs.uipath.com/overview/other/latest/overview/commercial-licensing-plans#flex-to-unified-pricing-user-license-mapping) in the Overview guide
:::
