You can use UiPath with a licensing plan to match your needs and budget.

## Plan overview

The following sections provide a high-level description of the licensing plans available in the Unified Pricing model.

### Application Testing Standard Trial

The Application Testing Standard Trial plan helps you explore how UiPath supports application testing at scale. With this trial you can:

* Use App Tester and App Test Developer licenses to manage most of the testing lifecycle, from test case creation, to execution, and analysis.
* Run tests with App Testing robots to validate application behavior in real environments.
* Leverage Autopilot and agentic capabilities to generate, maintain, and optimize tests.
* Try the extended platform capabilities that come with a paid Application Testing Standard plan.

The evaluation period for the Application Testing Standard Trial is limited to 60 days. After the trial expires, your organization is downgraded to Free.

### Application Testing Standard

The Application Testing Standard plan provides the following benefits:

* Use App Tester and App Test Developer licenses to manage most of the testing lifecycle, from test case creation, to execution, and analysis.
* Run tests with App Testing robots to validate application behavior in real environments.
* Leverage Autopilot and agentic capabilities to generate, maintain, and optimize tests.
* Discover testing opportunities across SAP systems using Change Impact Analysis, which assesses how proposed changes may affect your SAP system and helps you mitigate risks proactively.
* It provides core governance, security, enterprise scale capabilities to enable IT to manage your application testing operations.

### Application Testing Enterprise

The Application Testing Enterprise plan provides the following benefits:

* It provides access to Test Manager Premium Connectors to expand the integration with your ALM (Application Lifecycle Management) tools.
* It provides industry-leading governance, security, and enterprise scale capabilities, such as:
  + Customer Managed Keys: Encrypt your organization data stored across platform using UiPath managed keys or bring your own key to manage the data encryption.
  + External credential store: Offers the ability to configure Orchestrator to access external credential stores. You can store and retrieve sensitive data, such as Robot credentials and credentials assets, when needed.
  + Bring your own LLM: If you already have a paid subscription for Azure OpenAI, Anthropic, or Gemini, then you can use UiPath's AI capabilities on your subscription, instead of the one offered by UiPath.
  + Elastic Robot Orchestration: Allows you to automatically scale your unattended robots by allowing UiPath to scale and manage your robots for you, in the cloud.
  + CI/CD capabilities in Automation Ops: Enable continuous integration and deployment using the built-in Pipelines feature in Automation Ops, or through seamless integration with third-party CI/CD systems for more flexibility and control.
