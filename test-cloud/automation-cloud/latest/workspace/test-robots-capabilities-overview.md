## App Testing robots

You can use App Testing robots to run unattended automation for both development and testing purposes, only when working with Test Cloud. This includes working with workflows, activities, and various applications.

The license enables activity-level unit testing and allows multiple points of verification. This helps ensure the effectiveness of your automated tasks. It also provides granular reporting of test case results, allowing integration with third-party tools for a comprehensive overview of your testing efforts.

App testing robots can execute both application test cases and processes for non-production environments only.

For more information on App Testing robots, visit [App Testing](https://docs.uipath.com/orchestrator/automation-cloud/latest/user-guide/licensing#app-testing).

## Testing robots

You can use Testing robots for running unattended automations for both development and testing purposes. This includes working with workflows, activities, and various applications. Testing robots offer options that you can reuse across different projects.

The license enables activity-level unit testing and allows multiple points of verification. This helps ensure the effectiveness of your automated tasks. It also provides granular reporting of test case results, allowing integration with third-party tools for a comprehensive overview of your testing efforts.

Testing robots can execute both test cases and RPA or automation processes for non-production environments only.

To learn everything about robots, visit the official [UiPath®'s Robot documentation](https://docs.uipath.com/robot/standalone/latest/user-guide/introduction).
