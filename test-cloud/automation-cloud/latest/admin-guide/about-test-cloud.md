Test Cloud is an enterprise cloud offering designed for application testers. Test Cloud is specialized for application testing, surfacing UiPath® services that help you build a robust test portfolio. Given the platform capabilities inherited from Automation Cloud, Test Cloud offers a strong automation and testing ecosystem for managing all your testing operations: from automating tests, to distributing them, executing, and managing, you can perform all of these things in the context of a customizable cloud organization.

For more information on the Test Cloud platform capabilities, visit [About Automation Cloud](https://docs.uipath.com/automation-cloud/automation-cloud/latest/admin-guide/introduction).

## Key benefits

Key benefits of Test Cloud include the following:

* Robust platform capabilities: Offers capabilities such as data encryption, TLS protocol, integration assurance, access control, and secure images.
* Broad automation capabilities: Supports various types of automation including Citrix, Web, Desktop, SAP, Mainframe, API, and Mobile, catering to diverse testing requirements.
* Streamlined integration: Offers seamless CI/CD integration, version control, and distributed execution, boosting your development environment.
* Efficient management tools: Provides features for coherent requirements and test data management, enhancing operational efficiency.
* Detailed insights: Delivers comprehensive analytics and reporting tools for a well-rounded understanding of testing performance.
* Flexible testing methods: Supports data-driven testing with data directly uploaded into integrated UiPath® services, like Data Service and Orchestrator, or data generated with Autopilot<sup>TM</sup>. It also supports manual testing methods.
* Advanced discovery tools: Includes functionalities for test mining and discovery, helping identify areas for testing and ensuring that your applications uphold desired standards.
