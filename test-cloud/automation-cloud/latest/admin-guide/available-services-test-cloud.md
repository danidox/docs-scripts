Test Cloud surfaces various UiPath® services designed to automate test automations, enhance collaboration, and allow you to test a large pool of applications:

* **Action Center**: UiPath Action Center helps you manage and resolve exceptions, errors, or decisions that require human intervention in your agentic workflows. It provides a central interface for handling these exceptions, allowing human workers to review, make decisions, and take actions as needed to ensure the successful completion of agentic processes. For more information, visit the [Action Center documentation](https://docs.uipath.com/action-center/automation-cloud/latest).
* **Agents**: UiPath Agents is a development environment to build, evaluate, and publish enterprise Agents. It provides all of the necessary elements to create an agent which uses a large language model to interpret a natural language prompt, retrieve relevant context, call UiPath tools, and bring in humans as necessary to review output. For more information, visit the [Agents documentation](https://docs.uipath.com/agents/automation-cloud/latest/user-guide/about-agents).
* **AI Computer Vision**: UiPath AI Computer Vision is a machine-learning-based method used to visually identify all the UI elements on a computer screen and interact with them via UiPath Robots, simulating human interaction. It does not require or use the underlying properties of applications, but only the aspect and relationship of various screen elements. For more information, visit the [AI Computer Vision documentation](https://docs.uipath.com/ai-computer-vision/automation-cloud/latest).
* **Apps**: UiPath Apps is a low-code application development platform that enables you to build and share enterprise-grade custom applications that deliver engaging user experiences. Using Apps, you can quickly build custom business applications that connect to data in any underlying cloud or on-premises system using the power of automation. For more information, visit the [Apps documentation](https://docs.uipath.com/apps/automation-cloud/latest).
* **Automation Cloud Robots - Serverless**: UiPath Automation Cloud Robots - Serverless make it easy to run background automations without worrying about the necessary infrastructure. They provide you with complete freedom from provisioning, managing, maintaining, and scaling any underlying infrastructure. UiPath handles all the work behind the scenes so you do not have to deal with containers, virtual machines, or physical servers. For more information, visit the [Automation Cloud Robots - Serverless documentation](https://docs.uipath.com/orchestrator/automation-cloud/latest/user-guide/automation-cloud-robots-serverless).
* **Automation Cloud Robots - VM**: UiPath Automation Cloud Robots - VM is a hassle-free solution that streamlines the process of setting up the necessary infrastructure for running automations. It provides ready-to-use virtual machines where you can run automations. You just need to configure the virtual machine based on your need, and it is all set to run your tasks. For more information, visit the [Automation Cloud Robots - VM documentation](https://docs.uipath.com/orchestrator/automation-cloud/latest/user-guide/automation-cloud-robots-vm).
* **Automation Ops**: UiPath Automation Ops is a tool that enables you to manage and implement governance policies based on user profiles, and to manage the feeds available in your organization. For more information, visit the [Automation Ops documentation](https://docs.uipath.com/automation-ops/automation-cloud/latest).
* **Autopilot for Everyone**: The conversational version of Autopilot, available in UiPath Assistant, allows business users to run automations, query data, analyze images and files, and leverage generative AI capabilities through an intuitive chat interface. For more information, visit the [Autopilot for Everyone documentation](https://docs.uipath.com/autopilot/other/latest/user-guide/autopilot-for-everyone-feature-comparison).
* **Data Fabric**: UiPath Data Fabric is a persistent data storage service that brings no-code data modeling and storage capabilities to your test automation projects, allowing you to perform data-driven testing. For more information, visit the [Data Fabric documentation](https://docs.uipath.com/data-service/automation-cloud/latest).
* **Integration Service**: UiPath Integration Service is the component of the UiPath Platform that makes the automation of third-party applications much easier by standardizing authorization and authentication, helping you manage API connections, and also allowing faster integration into SaaS platforms. By using prebuilt APIs, the development of automation projects in UiPath Studio is more simple and more consistent, regardless of the third-party applications you are automating. For more information, visit the [Integration Service documentation](https://docs.uipath.com/integration-service/automation-cloud/latest).
* **Insights**: UiPath Insights is a tool that serves as a platform for data modeling and analytics using a combination of available business metrics and operational insights into your automations and robot performance. For more information, visit the [Insights documentation](https://docs.uipath.com/insights/automation-cloud/latest).
* **Maestro**: UiPath Maestro combines robotic and agentic workflows to seamlessly integrate human, AI, and robotic interactions, driving hyper-efficient business processes. With its robust capabilities, Maestro empowers businesses to optimize their operations at scale while simplifying complex workflows. For more information, visit the [Maestro documentation](https://docs.uipath.com/maestro/automation-cloud/latest/user-guide/introduction-to-maestro).
* **Orchestrator**: UiPath Orchestrator is a tool for streamlining the deployment, monitoring, and governance of your test automation processes and App test robots, giving you flexibility to schedule and trigger these processes as per your requirements. It also supports the management of transactional data, often used in automating business processes. For more information, visit the [Orchestrator documentation](https://docs.uipath.com/orchestrator/automation-cloud/latest).
* **Solutions**: UiPath Solutions provides an out-of-the-box means to manage the lifecycle of your solutions, once built. This set of capabilities comes into play once you are done developing your solutions, and you want to promote them to a new environment. For more information, visit the [Solutions documentation](https://docs.uipath.com/solutions-management/automation-cloud/latest).
* **Studio Web**: UiPath Studio Web enables you to build and test cross-platform automations across online business apps and services. All you need to start using Studio Web is a web browser. You can access it from any operating system, it requires no installation, it has no IT dependencies, and it is always up to date. For more information, visit the [Studio Web documentation](https://docs.uipath.com/studio-web/automation-cloud/latest).
* **Task Mining**: UiPath Task Mining offers insights into ongoing processes on your team's desktops, uncovering potential areas for automation and process improvement. This tool logs desktop data for each user action and uses AI to analyze it, thereby, giving you actionable insights for identifying automation ideas. For more information, visit the [Task Mining documentation](https://docs.uipath.com/task-mining/automation-cloud/latest).
* **Test Manager**: UiPath Test Manager is a test management tool for enterprise testing. It integrates into your ecosystem and adapts to your way of working, whether you use ServiceNow, SAP Solution Manager, Jira, Azure DevOps, or other ALM tools. For more information, visit the [Test Manager documentation](https://docs.uipath.com/test-manager/automation-cloud/latest).
