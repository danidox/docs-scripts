## November 10, 2025

**Build number: 2026.0.179**

### Support for EntraID users with multifactor authentication (MFA)

You can now use an unattended robot with EntraID users that have multifactor authentication enabled. Refer to [Using EntraID users with multifactor authentication (MFA) for unattended robots](https://docs.uipath.com/robot/standalone/latest/admin-guide/using-entraid-users-with-multifactor-authentication-mfa-with-robots).

### Bug fixes

* Ocasionally, the Robot would require to sign in again due to an incorrect use of the authentication token.
* This version brings several performance fixes.
