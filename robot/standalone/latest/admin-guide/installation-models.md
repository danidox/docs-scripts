UiPath offers solutions to fit your system requirements across various operating systems. For Windows users, UiPath provides two designated installers. Mac users can install the Robot in User Mode, along with the Assistant, on systems running version 10.15 (Catalina) or later. For Linux users, the Robot installation needs a Docker environment, a network connection to the Orchestrator, and the client credentials of the machine template.
