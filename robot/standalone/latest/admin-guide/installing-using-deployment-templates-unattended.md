You can deploy UiPath Robot through one of the following teamplates:

* [AWS Marketplace](https://docs.uipath.com/robot/standalone/latest/admin-guide/unattended-aws-deployment#aws-marketplace-deployment)—You can use Service Mode Robots to automatically run tasks on Amazon EC2.
* [Azure Marketplace](https://docs.uipath.com/robot/standalone/latest/admin-guide/unattended-azure-marketplace-deployment#azure-marketplace-deployment)—You can start using UiPath solutions directly from the Azure portal.
* [GCP Marketplace](https://docs.uipath.com/robot/standalone/latest/admin-guide/gcp-marketplace-deployment-unattended#gcp-marketplace-deployment)—You can set up the UiPath Robot directly from the marketplace.
