**Release date: July 21, 2025**

This patch for the 2024.10 version of Autopilot for Everyone contains a security fix.

We strongly advise to upgrade to the latest version, which you can automatically install from the Autopilot for Everyone Admin experience.

To keep using the 2024.10 release, you must install this patch manually, as it contains a security fix:

1. Go to **Resource Center** in your Automation Cloud organization.
2. Look for the 2024.10.15 Autopilot for Everyone artifact and download it to your device.
3. Once downloaded, switch to **Solutions**: **Orchestrator** > **Tenant** > **Solutions**, and upload the 2024.10.15 solution package. Read the [complete procedure](https://docs.uipath.com/solutions-management/automation-cloud/latest/user-guide/deploying-a-solution).
4. Select the **Deployments** tab.
   1. For the AutopilotForEveryone entry, from the contextual menu, select **Upgrade/Downgrade**.
   2. Select the **24.10.15** package version.
   3. Select **Continue**.
   4. In the next window, select **Deploy**.
5. Confirm the upgrade: the Autopilot automation version should be to **24.10.15**.
