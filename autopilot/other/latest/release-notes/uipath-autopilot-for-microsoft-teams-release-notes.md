## Conversational Agents in Autopilot for Teams

**Release date: September 22, 2025**

If you have access to the preview of conversational agents in UiPath, you can now start and manage chats tailored to your needs.

* Use **Switch Agent** from the **View Prompts** drop-down to move between agents or return to **Autopilot**.
* Be aware that only one agent is supported at a time in Teams. Conversations do not persist when switching back to a previously used agent.

## UiPath Autopilot for Microsoft Teams app

**Release date: November 6, 2024**

We are excited to announce the UiPath Autopilot for Microsoft Teams app.

The UiPath Autopilot for Microsoft Teams app:

* Helps you find and execute UiPath Autopilot automations within Microsoft Teams.
* Understands the context of your work.
* Provides an interactive chat experience powered by generative AI.

You can find the UiPath Autopilot for Microsoft Teams app in the Microsoft Teams marketplace, in the Copilot Extensions tab.

For more details, refer to our [UiPath Autopilot for Microsoft Teams](https://docs.uipath.com/autopilot/other/latest/overview/autopilot-plug-ins) documentation.
