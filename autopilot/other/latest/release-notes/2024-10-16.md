**Release date: February 20, 2026**

## LLM updates

Version 2024.10.16 of Autopilot for Everyone now uses updated models: **Claude Sonnet 4.5** as the main model and **GPT-4.1 mini** as the fast model.

## High-contrast theme support

We've added support for high contrast Dark and Light themes to improve accessibility and visual clarity.

## Downloading version 2024.10.16

To keep using the 2024.10 release, you must install this patch manually:

1. Go to **Resource Center** in your Automation Cloud organization.
2. Look for the 2024.10.16 Autopilot for Everyone artifact and download it to your device.
3. Once downloaded, switch to **Solutions**: **Orchestrator** > **Tenant** > **Solutions**, and upload the 2024.10.16 solution package. Read the [complete procedure](https://docs.uipath.com/solutions-management/automation-cloud/latest/user-guide/deploying-a-solution).
4. Select the **Deployments** tab.
   1. For the AutopilotForEveryone entry, from the contextual menu, select **Upgrade/Downgrade**.
   2. Select the **24.10.16** package version.
   3. Select **Continue**.
   4. In the next window, select **Deploy**.
5. Confirm the upgrade: the Autopilot automation version should be to **24.10.16**.
