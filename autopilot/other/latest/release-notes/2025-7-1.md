**Release date: July 21, 2025**


:::note
This version is the feature baseline for version 2025.9.3 of Autopilot for Everyone, included in [Automation Suite 2.2510.0](https://docs.uipath.com/automation-suite/automation-suite/2.2510/release-notes/automation-suite-on-eks-aks-2-2510-0).

## Custom feedback routing

Autopilot for Everyone now allows routing your feedback data to a custom destination through a dedicated automation, called **SubmitAutopilotFeedback**. The automation considers the following type of feedback: Thumbs down, Thumbs up, and General feedback. To activate this functionality, organizations admins must:

* publish and provide a version of the **SubmitAutopilotFeedback** automation for their users, and
* enable the new **Send feedback via SubmitAutopilotFeedback** setting in Autopilot for Everyone Admin.

The feedback routing is in addition to sending your feedback to UiPath, if you have the **Send feedback to UiPath** setting on.

## Breaking change

If you hold an Automation Express, Autopilot Express, or Express licenses, you can now run only Out-of-the-Box (OOTB) automations in the Autopilot and Tool Automations folders and subfolders. You can no longer run automations in any other folder.

## Improvements

The error messaging experience in Autopilot for Everyone is now improved.
