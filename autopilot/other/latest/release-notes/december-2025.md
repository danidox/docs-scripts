### Autopilot chat joins UiPath Maestro™

Autopilot™ for Maestro is now generally available. Use natural-language prompts and documents to generate and refine BPMN models, write C# and JavaScript expressions, and get design guidance in a unified, streaming chat experience that handles lanes more safely.

[Discover the tools](https://docs.uipath.com/maestro/automation-cloud/latest/user-guide/autopilot-fo) Autopilot uses and start designing BPMN models faster, with less manual configuration.
