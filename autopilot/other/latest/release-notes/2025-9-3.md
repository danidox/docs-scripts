**Release date: November 11, 2025**

As of November 11, you can install and configure Autopilot for Everyone from the Admin sectionof your Automation Suite platform, under the AI Trust Layer card. Once set up, you’ll be able to benefit from its capabilities directly in Assistant.

Autopilot for Everyone provides over 100 pre-built UiPath automations, available as toolsets in UiPath Marketplace. You can use these automations as they are or build your own, relying on Autopilot to help you complete your daily tasks efficiently.

To use Autopilot for Everyone in Automation Suite, you must provide an LLM subscription for the models Autopilot uses — either Claude 3.7 Sonnet or GPT-4o-mini.

Note that installing Autopilot for Everyone in Automation Suite comes with certain usage limitations. For more information, read the [Autopilot for Everyone feature comparison](https://docs.uipath.com/autopilot/other/latest/user-guide/autopilot-for-everyone-feature-comparison).
