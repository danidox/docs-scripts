## Conversational Agents in Autopilot for Slack

**Release date: September 22, 2025**

If you have access to the preview of conversational agents in UiPath, you can now start and manage chats tailored to your needs.

* Select **New Chat**, then choose an agent from the **Select Conversational Agents** list.
* To switch to another agent or return to **Autopilot**, go to the **Home** tab or start a new chat with **+ New Chat**.

## UiPath Autopilot integration for Slack

**Release date: February 19, 2025**

UiPath Autopilot now integrates directly into Slack, providing AI-powered automation to resolve tasks without switching applications. Engage Autopilot via text commands and buttons, receiving structured responses directly in chat. Robust security is maintained through UiPath Platform standards.

Find out more in [our documentation](https://docs.uipath.com/autopilot/other/latest/overview/uipath-autopilot-for-slack).
