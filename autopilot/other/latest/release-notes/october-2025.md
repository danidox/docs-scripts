## October 30, 2025

### Autopilot chat joins Studio

Autopilot chat is now available in Studio, starting with version 2025.10.1, where it takes the form of an advanced workflow assistant. You can ask in plain-language to create, debug, and optimize your automation workflows.

Autopilot uses specialized tools to assist you with your development tasks and it provides accelerated development by:

* generating workflow components and suggesting activities,
* optimizing workflows by diagnosing errors, suggesting code improvements, and validating against best practices,
* reducing manual effort to allow for strategic focus.

Autopilot improves efficiency and quality in automation and tests creation.

[Discover the tools](https://docs.uipath.com/autopilot/other/latest/user-guide/autopilot-one-in-studio-desktop) Autopilot uses and start developing automations.
