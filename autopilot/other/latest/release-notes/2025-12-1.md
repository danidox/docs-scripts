**Release date: January 16, 2026**

## Improved process list updates

Improved process list refresh behavior to prevent load spikes when process information is updated in Orchestrator.

## Non-ASCII characters support

RPA tools can now run correctly when non-ASCII characters are used in parameter names.

## Accessibility improvements

Several accessibility improvements have been made in this release to enhance usability and compliance across the Autopilot for Everyone experience.

## Bug fixes

* Fixed issues related to Action Center integration.
* Improved Integration Service connection handling.
