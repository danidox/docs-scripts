To chat in a specialized context, you can select a [specialized Autopilot](https://docs.uipath.com/autopilot/other/latest/user-guide/specialized-autopilot#specialized-autopilot), predefined by your administrator.

1. Launch Autopilot for Everyone. The general-purpose chat window opens.
2. From the Autopilot for Everyone dropdown menu, select the specialized instance. The context grounding indexes, data sources, and advanced settings for the selected Autopilot are loaded to the chat context.
3. Prompt the specialized Autopilot.

You can select a specialized Autopilot mid-conversation, thus changing the chat context.

Starting a new chat or selecting a previous session from the chat history reverts the context to the general-purpose Autopilot for Everyone.
