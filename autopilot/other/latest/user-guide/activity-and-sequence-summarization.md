Autopilot™ can summarize the inner workings of activities and sequences by creating clear, meaningful names that take into account the elements that define the activity or sequence. This capability is particularly useful for improving the readability of large workflows, which can be hard to review or maintain.

To learn how Autopilot can help you better describe your automations, see:

* [Managing the activities in a project](https://docs.uipath.com/studio-web/automation-cloud/latest/user-guide/managing-activities) in the Studio Web guide.
* [The Designer Panel](https://docs.uipath.com/studio/standalone/2024.10/user-guide/the-user-interface#the-designer-panel) in the Studio guide.
* [The Autopilot Panel](https://docs.uipath.com/studio/standalone/2024.10/user-guide/the-user-interface#the-autopilot-panel) in the Studio guide.
