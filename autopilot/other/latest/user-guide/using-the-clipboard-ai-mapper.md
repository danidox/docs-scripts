With **Paste with Mapper**, the Clipboard AI mapper displays the table as the copied source and the application selected in the paste menu as the target. You continue with the paste process within the mapper. Once the paste operation is complete, close the mapper.

Read more details about the [Data mapper](https://docs.uipath.com/clipboard-ai/standalone/latest/user-guide/data-mapper).
