:::

Clipboard AI integration with Autopilot for Everyone allows you to perform AI-powered copy-paste operations from within the chat.

The integration allows the following pasting scenarios:

* Prompting Autopilot to render a table from a document, such as an invoice, receipt, or purchase order, then pasting the rendered data into an Excel spreadsheet.
* Prompting Autopilot to copy data from a multi-line table, then pasting the rendered data into a web form.
* Prompting Autopilot for a response, then pasting the response.
* Uploading a file to Autopilot, then pasting the extracted data.
