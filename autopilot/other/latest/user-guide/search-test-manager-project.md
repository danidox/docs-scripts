With the help of the Autopilot<sup>TM</sup> search, you can search any test object within a project using natural language. If you are not sure what to search for, you can use one of the example search queries that Autopilot provides. Additionally, after returning the results, Autopilot allows you to perform actions on the resulted objects.

For information on using the search, visit [Searching with Autopilot](https://docs.uipath.com/test-suite/automation-cloud/latest/user-guide/searching-with-autopilot).
