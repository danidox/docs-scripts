The Clipboard AI toolbar offers extended functionality beyond the ones available in the chat experience.

You can access the Clipboard AI toolbar through:

* The **Launch Clipboard AI** button in Autopilot for Everyone version 2025.4+
* The **Paste in** option under tables:
  + **Paste with Mapper**
  + **Send to Clipboard AI**
* The **Paste in** option after a prompt response or a file upload
* The Assistant launchpad tile, which opens the Clipboard AI toolbar.
