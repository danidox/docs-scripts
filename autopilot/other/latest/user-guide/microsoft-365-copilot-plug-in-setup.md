To enable the Microsoft 365 Copilot Plug-in, make sure to install the UiPath Autopilot for Microsoft Teams app before you begin this procedure. Refer to [Setup](https://docs.uipath.com/autopilot/other/latest/user-guide/uipath-autopilot-for-microsoft-teams-setup) for more information.

To access Copilot features, you must have Copilot enabled in your Microsoft 365 tenant.

To enable the Microsoft 365 Copilot Plug-in, take the following steps:

1. Access **Teams,** then select **Chat.**
2. Select **Copilot.**
3. Under the **Message Copilot** field, select the **Plug-ins** button.
4. To enable the plug-in, select **UiPath Autopilot**.
