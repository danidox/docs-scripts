Data sources provide relevant context to Autopilot for Everyone to answer user queries. The decision to query a certain data source is based on its description. To answer a query, Autopilot has the ability to cross-reference up to six data sources.

Autopilot has access to several default data sources:

* Autopilot FAQ (*)
* Google (**)
* Wikipedia (**)

(*) Autopilot FAQ is a customizable data source used to provide information about Autopilot. You can edit it to fit your organization specific needs.

(**) Autopilot for Everyone uses Google and Wikipedia to find publicly available information. To stop these external searches, organization admins need to remove them from the user-accessible folder, or business users can disable them from [Autopilot Settings > Advanced settings > Data sources](https://docs.uipath.com/autopilot/other/latest/everyone-user-guide/interacting-with-autopilot-for-everyone#autopilot-settings-in-assistant).
