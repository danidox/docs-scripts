Once the [installation of Autopilot for Everyone](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/installing-autopilot-for-everyone) is complete, open UiPath Assistant standalone and sign in to the tenant that hosts Autopilot.

For usage in Assistant web, open Assistant from the left rail in Automation Cloud™.

1. Assistant installs Autopilot, and the **Autopilot** tab appears to the left of the **Home** tab.
2. Autopilot displays the welcoming screen, where you may provide data about yourself, such as **Role**, **Department**, **Location**, and you can add a profile picture. You can always adjust this information from the **Settings** section of Autopilot.
3. Select **Next** to enter the chat landing page.

## Autopilot for Everyone landing page

Every **New chat** request opens the landing page of Autopilot for Everyone, with the following details:

   ![Autopilot for Everyone landing page](/images/autopilot/autopilot-autopilot-for-everyone-landing-page-558893-f5522d9a.webp)

1. Title—helps you distinguish between general-purpose Autopilot for Everyone, and specialized autopilot, which have different names.
2. Description—provides a short description of the context of the selected Autopilot.
3. The chat box—allows you to input your prompts. It also allows uploading attachments, or capturing screenshots, or active windows in the chat, if your admin [enabled the **File upload** setting](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/advanced-settings) for your tenant.
4. Starting prompts categories—allows you to switch between categories of starting prompts and access all prompts within each. Additionally, it allows you to save frequently used starting prompts and access them from the dedicated **Saved** tab.
