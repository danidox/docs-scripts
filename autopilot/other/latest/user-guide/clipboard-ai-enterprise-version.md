Autopilot for everyone installs the latest Enterprise version of Clipboard AI.

Clipboard AI in the Autopilot chat experience facilitates pasting tables into a selected application. Learn how to [paste tables with Clipboard AI](https://docs.uipath.com/autopilot/other/latest/everyone-user-guide/pasting-operations-with-clipboard-ai).

To leverage the extended functionality of Clipboard AI, you need to access the Clipboard AI toolbar:

* The **Launch Clipboard AI** button in Autopilot for Everyone version 2025.4+
* The **Paste in** option under tables:
  + **Paste with Mapper**
  + **Send to Clipboard AI**
* The **Paste in** option after a prompt response or a file upload
* The Assistant launchpad tile, which opens the Clipboard AI toolbar.

Refer to the [Clipboard AI documentation](https://docs.uipath.com/clipboard-ai/standalone/latest/user-guide/introduction) for more details.
