Make sure you enabled the Teams Search Messaging Extension before beginning this procedure. Refer to [Setup](https://docs.uipath.com/autopilot/other/latest/user-guide/teams-search-extension-setup) for more information.

1. From Teams, access **Chat.**
2. To choose an extension, select the **+ icon** to the right of the **Type a message** field.
3. To use the Teams Search Messaging Extension, search for, then select the **UiPath Autopilot** option in the extensions menu.
4. To search for an automation, enter a keyword in the **Search bar.** The extension retrieves a list of results.
   :::note
   In addition to keywords, you can search by:
   * The tag or tags linked to an automation.
   * The name of the folder in which you saved the automation.
   :::
5. Select the automation you want to use. The automation preview window opens.
6. To proceed with the selected automation, select the **Send** button.
7. Enter inputs for the automation arguments. When you are ready, select **Run automation.** The job status window opens.
8. To get the latest job status, select the **Refresh** button.
   :::note
   The job status can be one of several states:
   * **Pending -** the job is in a queue and is waiting to be executed by a robot.
   * **Running -** the robot is executing the job.
   * **Faulted -** job execution has failed due to a workflow error.
   * **Successful** - job execution is complete.
   :::
