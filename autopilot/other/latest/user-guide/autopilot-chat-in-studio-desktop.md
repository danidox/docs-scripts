The Autopilot chat in Studio provides agentic capabilities to assist you in your automation development environment for various tasks. These capabilities include:

* understanding complex intent
* planning and reasoning
* executing actions
* monitoring and fixing issues on the spot

## Connected sources

The Autopilot chat in Studio leverages a range of built-in tools to support your automation work. By default, all tools are enabled.

* **RPA Workflow Assistant**—Creates, edits, summarizes, or explains RPA workflows (.xaml files). Expand the tool to disable it or to enable **Allow Edit Workflow**.
  :::important
  Use the **Allow Edit Workflow** setting with caution, as it allows Autopilot to interfere with your workflow.
  :::
* **Web Search**—Answers questions that require updated, external information.
* **UiPath Documentation Search**—Retrieves content from UiPath official documentation.
* **Web Reader**—Extracts and summarizes content from public web pages.
* **Run Workflow**—Executes or debugs a workflow within UiPath Studio Desktop.
* **Read File**—Reads the contents of files within your current project.
* **List Directory**—List the files and subdirectories within a given directory.
* **Get Project Structure**—Displays the file structure of the current project.
* **Write File**—Creates new files or write content into existing ones within your project.
* **Edit File**—Makes targeted modifications to the content of existing files.
* **Move File**—Moves files between directories or rename them within the project structure.
* **Search File**—Searches for specific text patterns or content within files across your project.
* **Delete File**—Deletes files from your project.
* **Lint File**—Analyze files for coding errors, style issues, and potential bugs, and provides suggestions for improvement or automatic fixes.
* **Retrieve Code Examples & Definitions**—Provide code examples, API documentation, and definitions for code elements.
* **Get Errors**—Retrieves errors for the current file or entire project.
* **Get Type Definitions**—Provides type definitions for variables, properties, and methods in C# files.
* **Get Quick Fixes**—Suggests quick fixes for identified errors in C# files.
* **Run Powershell Command**—Executes PowerShell commands for advanced operations.
* **Code Generation Prerequisites**—Provides C# code examples, API documentation, and entity information for generating and understanding C# code.
* **Retrieve Manual Test Cases**—Retrieves information about manual test cases from Test Manager. This is valuable for test automation processes.
* **Retrieve Manual Test Steps**—Retrieves the detailed steps associated with specific manual test cases. This provides granular information for automating or analyzing tests.

## For RPA and coded workflows

Autopilot helps you build and edit RPA workflows in Studio Desktop using natural language.

Use the chat to:

* Create or edit RPA workflows by describing the desired automation steps.
* Generate C# code snippets, classes, or coded workflows based on your requests, leveraging UiPath-specific APIs.
* Configure activities and code logic, including input/output data and control flow.
* Understand what existing activities or code blocks do.
* Improve workflow logic with better structure or error handling.
* Analyze files for linting errors and apply quick fixes.
* Debug and run workflows directly from the chat.
