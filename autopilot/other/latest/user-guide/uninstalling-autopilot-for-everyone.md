Uninstalling Autopilot for Everyone removes the solution from the **Solutions** tab in Orchestrator, resulting in end users no longer being able to interact with Autopilot within UiPath Assistant. After uninstalling, we recommend to also disable the Autopilot widget within the Assistant, to prevent the display of a **Learn more** screen.

To uninstall Autopilot for Everyone:

1. Navigate to the **Admin** section in your Automation Cloud™ organization.
2. Select the **AI Trust Layer** card, and switch to the **Autopilot for Everyone** tab.
3. From the **Tenant *** dropdown menu, select the tenant from which you want to uninstall Autopilot for Everyone.
4. Once you select a tenant, the **Uninstall** button becomes active. Select it to uninstall Autopilot for Everyone.
5. Disable the Autopilot widget as [instructed in the Automation Ops documentation](https://docs.uipath.com/automation-ops/automation-cloud/latest/user-guide/settings-for-assistant-policies#widgets).
