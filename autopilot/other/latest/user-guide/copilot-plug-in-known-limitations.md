The current known limitations of the Microsoft 365 Copilot Plug-in are as follows:

* The Microsoft 365 Copilot Plug-in does not automatically update the status of an automation. To see the up-to-date status of an automation, select the **refresh** button.
* The Microsoft 365 Copilot Plug-in only supports automations with the Autopilot label.
* The Microsoft 365 Copilot Plug-in cannot pre-fill any data.
* The Microsoft 365 Copilot Plug-in only supports unattended execution.
* To change the active tenant, access the [Teams Search Messaging Extension](https://docs.uipath.com/autopilot/other/latest/user-guide/teams-search-extension-setup).
