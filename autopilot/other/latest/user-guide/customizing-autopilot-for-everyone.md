To customize Autopilot for end users, access the **Admin** section in your Automation Cloud® organization. Navigate to the **AI Trust Layer** card, select the **Autopilot for Everyone** tab, then choose your tenant. The following settings become available for configuration:

* [Starting prompts](https://docs.uipath.com/autopilot/other/latest/user-guide/stating-prompts#starting-prompts)—predefined prompts aimed to provide effective instructions for executing common tasks.
* [Context Grounding](https://docs.uipath.com/autopilot/other/latest/user-guide/context-grounding#context-grounding)—allows you to enable existing Context Grounding indexes in Autopilot for Everyone, which helps perform searches to complete user queries.
* [Tools](https://docs.uipath.com/autopilot/other/latest/user-guide/automation-properties#tools)—control how automations tools are configured in Autopilot for Everyone.
* [Advanced settings](https://docs.uipath.com/autopilot/other/latest/user-guide/advanced-settings)—controls various functionality for Autopilot for Everyone, such as file uploads, user notes, or setting a chat history limit.
* [License settings](https://docs.uipath.com/autopilot/other/latest/user-guide/license-settings#license-settings)—controls the consumption of AI units for Autopilot actions.
