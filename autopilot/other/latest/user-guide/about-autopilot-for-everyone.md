Autopilot for Everyone is UiPath AI assistant built to help business users with daily tasks. It has the following capabilities:

* Generative AI—creates content, analyzes data, and generates insights
* Context Grounding—searches Context Grounding indexes to answer user queries
* Automations—recommends and runs UiPath automations
* Clipboard AI <sup>1</sup>—performs AI-powered copy-paste operations <sup>1</sup> ClipboardAI capabilities are not availabe for Public Sector customers.

## Limitations

* You can only upload the following file extensions:
  + DOCX
  + PPTX
  + CSV
  + JSON
  + TXT
  + PDF
  + PNG
  + JPG/ JPEG
  + XML/XAML
* You can upload files up to 25 MB.
* You can only run RPA-process types, as Autopilot for Everyone does not offer support for Agents or Maestro workflows.
* You can perform a single Clipboard AI paste from a table in Autopilot for everyone. However, you can still access the full capabilities of Clipboard AI by launching the Clipboard AI toolbar.
* The Clipboard AI functionality works exclusively on Windows operating systems.
* Overlapping or ambiguous descriptions for automations or Context Grounding indexes can lead Autopilot to suggest or carry out actions that may not be intended.

## Limitations for Automation Cloud™ Public Sector customers

* ClipboardAI capabilities are unavailable.
* The chat experience does not provide a [general feedback button](https://docs.uipath.com/autopilot/other/latest/everyone-user-guide/giving-feedback).
* Installing Autopilot for Everyone only installs the Autopilot process in the Autopilot folder, [compared to installing processes related to default automations too](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/installing-autopilot-for-everyone).
* [The **Tools bundle** tab](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/automation-properties#tools-bundle) contains instructions for manual download and install of available out-of-the-box automations.
