The UiPath Autopilot for Microsoft Teams app allows you to discover and run UiPath Autopilot automations within Microsoft Teams.

The UiPath Autopilot for Microsoft Teams app contains the following components:

* [The 1:1 chat with Autopilot for Teams app.](https://docs.uipath.com/autopilot/other/latest/user-guide/1-1-chat-with-autopilot-for-teams)
* [The Teams Search Messaging Extension.](https://docs.uipath.com/autopilot/other/latest/user-guide/teams-search-messaging-extension)
* [The Microsoft 365 Copilot Plug-in.](https://docs.uipath.com/autopilot/other/latest/user-guide/microsoft-365-copilot-plug-in)

## Prerequisites

Before you begin:

1. Ensure access to a UiPath Automation Cloud organization.
2. Verify permissions to see and run unattended automations.
3. Ensure installation of Autopilot for Everyone in the tenant where you want to connect.
4. Confirm you have access to Autopilot for Everyone.
5. Tag the automations you want to run in Microsoft Teams with the **Autopilot** label. Only automations tagged with the **Autopilot** label are visible in Microsoft Teams.


:::important
To enable Autopilot for Everyone to correctly route your requests within Microsoft Teams, automations must be tagged with the **Autopilot** label.
:::
