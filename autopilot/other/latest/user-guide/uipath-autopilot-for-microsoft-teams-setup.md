To install the UiPath Autopilot for Microsoft Teams app, take the following steps:

1. Sign in to **Microsoft Teams**.
2. Select **Apps.**
3. Search for **UiPath Autopilot.**
4. Select the app from the search results.
5. Select **Add.**
