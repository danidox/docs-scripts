---
title: "Sending the table to Clipboard AI"
visible: true
slug: "sending-the-table-to-clipboard-ai"
---

From the **Paste in** dropdown menu, you can select **Send to Clipboard AI**. This opens the entire Clipboard AI toolbar, setting the Autopilot table as the copied source.

From this point, you can paste the copied information anywhere you want using the complete Clipboard AI toolbar functionality. You are free to continue your work in Clipboard AI as if you are using a standalone app.
