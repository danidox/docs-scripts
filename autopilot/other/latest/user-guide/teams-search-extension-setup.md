Make sure to install the UiPath Autopilot for Microsoft Teams app before you begin this procedure. Refer to [Setup](https://docs.uipath.com/autopilot/other/latest/user-guide/uipath-autopilot-for-microsoft-teams-setup) for more information.

To enable the Teams Search Messaging Extension, take the following steps:

1. From Teams, access **Chat**.
2. To view the list of Teams extensions, select the **+** button**.**
3. Select **UiPath Autopilot**.
4. Sign in to your UiPath Automation Cloud organization.
   :::note
   When you first sign in to your Automation Cloud organization, the **UiPath Autopilot** plugin selects the first tenant by default. To change the active tenant, select the **additional options** button in the extension, then select **Change Tenant.** The selected tenant is saved and it persists across sessions.
   :::
