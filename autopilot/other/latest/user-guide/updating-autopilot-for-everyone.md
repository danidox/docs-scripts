visible: true

---

Autopilot constantly checks for the latest version. When a new version is available, the deployment section of Autopilot for Everyone also displays it.

To update your Autopilot for Everyone version:

1. Navigate to the **Admin** section in your Automation Cloud™ organization.
2. Select the **AI Trust Layer** card, and switch to the **Autopilot for Everyone** tab.
3. From the **Tenant *** dropdown menu, select the tenant for which you want to update Autopilot for Everyone.
4. Once you select a tenant, an update message and the **Update** button are displayed. Select **Update** to install the newest version of Autopilot for Everyone.

Updating to a newer version of Autopilot for Everyone preserves the configurations you made for the previous versions.


:::important
Review your configurations after updating the version, as new options may be available.
:::
