Autopilot™ can generate apps based on an existing PDF or image. Autopilot automatically digitizes the text within the file and builds the app based on its structure. This is particularly useful when working with legacy paper forms which you have scanned. You can also use an image or PDF to add a new page to an existing app.

![docs image](/images/autopilot/autopilot-docs-image-435954-e77026f1.webp)

Visit [Using a PDF or image to generate an app](https://docs.uipath.com/apps/automation-cloud/latest/user-guide/pdf-or-image-to-app) to learn how to create apps using PDFs or images.
