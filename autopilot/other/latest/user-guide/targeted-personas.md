## Organization administrators

To deploy and configure Autopilot for Everyone in your organization, administrators have a dedicated section in Automation Cloud where they can manage Autopilot for Everyone installations and configure the following Autopilot features:

* Starting prompts
* Context Grounding
* Tools
* Advanced settings
* License settings

Learn how to [manage](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/installing-autopilot) and [configure](https://docs.uipath.com/autopilot/other/latest/everyone-admin-guide/installing-autopilot#customizing-autopilot-for-everyone) Autopilot for Everyone.

## RPA developers

To build automations optimized for Autopilot for Everyone, RPA developers can use the template UiPath provides.

Learn how to [design Autopilot-relevant automations](https://docs.uipath.com/autopilot/other/latest/user-guide/designing-autopilot-automations#designing-relevant-automations-for-autopilot-for-everyone).

## Business users

To prompt and interact with Autopilot for Everyone, business users need to launch Assistant. Start your chat by selecting from predefined prompts, or input your own query.

Learn how to [interact with Autopilot for Everyone](https://docs.uipath.com/autopilot/other/latest/everyone-user-guide/interacting-with-autopilot-for-everyone).
