Using Autopilot™, you can generate apps using a natural language text prompt. Autopilot interprets your prompt and generates the desired app. Autopilot also provides a number of predefined text prompts for some common use cases. You can add a new page to an existing app using a text prompt too.

In addition, Autopilot can build VB expressions for use in your apps based on a text prompt:

![docs image](/images/autopilot/autopilot-docs-image-436003-536b25d8.webp)

To learn more about how Autopilot can help you accelerate your app development, visit [Using a text prompt to generate an app](https://docs.uipath.com/apps/automation-cloud/latest/user-guide/text-to-app) in the Apps guide.
