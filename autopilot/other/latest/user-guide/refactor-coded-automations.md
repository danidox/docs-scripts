Autopilot<sup>TM</sup> helps you to enhance your code using refactoring. If you select the code for which you want to perform refactoring, then right-click and select **Generate code**, you can offer Autopilot instructions for enhancing it. You can offer instructions to improve the selected code by identifying redundancies, simplifying complex methods, maintaining consistent coding standards, among others.
