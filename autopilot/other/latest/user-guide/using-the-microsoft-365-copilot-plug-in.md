Make sure you [enabled the Microsoft 365 Copilot Plug-in](https://docs.uipath.com/autopilot/other/latest/user-guide/microsoft-365-copilot-plug-in-setup).

To access Copilot features, you must have Copilot enabled in your Microsoft 365 tenant.

1. From the Teams sidebar, select **Copilot.**
   :::note
   The Copilot window displays the text **UiPath is on** if the Microsoft 365 Copilot Plug-in is enabled.
   :::
2. To interact with the Microsoft 365 Copilot Plug-in:
   * When you select **Try now,** a list of possible queries is displayed.
   * Write text in the **Message Copilot** field to prompt the AI.

You can find and run automations with the Autopilot label using the Microsoft 365 Copilot Plug-in.
