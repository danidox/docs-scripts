**Release date: August 26, 2025**

## Bug fixes

* You could not run workflows containing UI Automation activities on local Mac machines.
* Unexpected errors occurred when opening some projects containing Data Service activities if those projects were previously edited in Studio Desktop.

For details regarding this patch, refer to the Automation Suite on EKS/AKS release notes: [About this patch](https://docs.uipath.com/automation-suite/automation-suite/2024.10/release-notes/automation-suite-on-eks-aks-2024-10-4-patch-1#about-this-patch).

:::important
This fix will be included in all future releases starting with Automation Suite 2024.10.5. We recommend you wait for Automation Suite 2024.10.5 to get this fix.
:::
