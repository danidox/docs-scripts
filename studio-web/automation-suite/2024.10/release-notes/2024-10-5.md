**Release date: September 24, 2025**

## Bug fixes

* An *Execution Timeout Expired. The timeout period elapsed prior to completion of the operation or the server is not responding.* error occurred when importing or downloading some projects.
* A `System.InvalidOperationException: The Framework\InitState.xaml workflow cannot be found.` error occurred when using the **Test on local machine** option to debug a workflow that was included in a folder.
