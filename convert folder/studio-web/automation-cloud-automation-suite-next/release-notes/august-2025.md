## August 26, 2025

### Introducing Validation Control

The **Validation Control** is a powerful new UI component that brings human-in-the-loop document validation directly into your UiPath Web Apps. It allows you to display documents and extracted data, enabling business users to review, correct, and confirm fields in a clear and structured interface.

Validation Control gives you full flexibility to design the validation experience around your users and business rules. It eliminates the need to switch between interfaces, improves validation accuracy, and enhances automation continuity by embedding data review directly into your app.

You can edit single fields, table rows, and multi-value entries, with options to apply confidence scores or confirmation flags. The control is powered by the `ContentValidationData` object, which connects Document Understanding results to your app.

You can also take advantage of event-based behaviors, such as triggering automation when a field is updated or a document type is changed, giving you fine-grained control over how validation works for your users.

Learn more in [our documentation](https://docs.uipath.com/studio-web/automation-cloud/latest/user-guide/validation-control) and get started today.

## August 6, 2025

### Start building API workflows...

A new project type joins the UiPath Studio Web portfolio: API workflows.

Start building automations that integrate directly with the APIs of your favorite business applications, such as Salesforce, SAP, or Workday. This means faster execution, more reliable automations, and the ability to handle complex data with ease.

API workflows offer you direct connections to pre-built Integration Service connectors, built-in logic and error handling, and advanced data handling through JSON and XML processing capabilities.

Discover the basics of API workflows in, and hAPI automating!

### ... and keep the flow going with Autopilot chat

Autopilot chat transforms the way you interact with AI in the UiPath ecosystem. Now available for API workflows in Studio Web, it introduces a single conversational interface that stays with you throughout your task.

Whether you're building a process, refining a prompt, or troubleshooting an automation, Autopilot chat provides contextual support exactly when and where you need it. It remembers what you're working on, understands your intent, and offers relevant, custom suggestions.

By combining context retention, real-time interaction, and a unified workspace, Autopilot chat simplifies development, reduces repetition, and helps you get to better results, faster.

[Start using Autopilot. Chat](https://docs.uipath.com/autopilot/other/latest/overview/about-autopilot-chat) and chill.
