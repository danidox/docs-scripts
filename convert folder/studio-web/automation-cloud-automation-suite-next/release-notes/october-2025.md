## October 27, 2025

### Dynamic tabs in apps

Dynamic tabs introduce a new way to create tab navigation in apps. Instead of manually defining each tab, tabs are generated automatically at runtime based on a selected data source: list, table, or entity. This makes tab management flexible and data-driven.

To learn more, see.

### External events in apps

External events are here to bring custom, cross-window communication messages and secure, real-time data exchange between parent windows and embedded apps, allowing apps to trigger specific actions based on custom event types.

To enable this feature, UiPath Communication Driver, a lightweight communication library is now available to provide context sharing and communication between host systems and embedded apps created in Studio Web.

For more information, see.

### CX Companion template

We are pleased to announce the introduction of CX Companion, an app that helps customer care representatives work more efficiently. CX Companion can be embedded in host systems such as Salesforce and integrated in the workspace, enabling a single destination where support representatives can view, execute, and track UiPath automations, apps, and AI agents based on the context of incoming requests.

CX companion is available as a customizable solution template for Apps in Studio Web. A dedicated CX Companion plugin is available for integration with Salesforce, and a communication driver is available for download as an NPM package to facilitate integration with other third party systems.

For more information, see.
