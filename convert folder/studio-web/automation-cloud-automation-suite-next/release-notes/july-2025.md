## July 29, 2025

### Studio Web in the United States delayed update region

To ensure GxP regulatory compliance in life sciences, Studio Web is now available in the United States delayed update region. For more information, refer to [Delayed update organizations](https://docs.uipath.com/studio-web/automation-cloud/latest/release-notes/release-notes-2025#delayed-update-organizations).

## July 21, 2025

### Improvements

* Opening a solution now opens the last modified project within that solution.
* Finding and selecting solution resources in activities is now faster and easier.

  ![docs image](/images/studio-web/studio-web-docs-image-580108.webp)

### Bug fixes

You did not have permission to view Studio Web if your existing Automation Cloud account was added to a local group that was then assigned the Studio Web Contributor role.
