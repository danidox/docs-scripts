## 19 December 2024

### Improvements

In Web App Projects, you can now add the [Document Viewer](https://docs.uipath.com/apps/automation-cloud/latest/user-guide/document-viewer) control and use the [Upload Storage File](https://docs.uipath.com/activities/other/latest/workflow/upload-storage-file) and [Download Storage File](https://docs.uipath.com/activities/other/latest/workflow/download-storage-file) events.

### Bug fixes

An unexpected error sometimes occurred when navigating to Studio Web after signing in to UiPath Automation Cloud.
