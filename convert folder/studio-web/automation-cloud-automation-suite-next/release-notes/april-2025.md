## 30 April 2025

### Agentic processes are Generally Available (GA)

The experience of modelling and implementing agentic processes in Studio Web is now generally available. Refer to the [Agentic processes](https://docs.uipath.com/studio-web/automation-cloud/latest/user-guide/agentic-processes) section in the Studio Web user guide to start working with agentic processes.

For more information on how to orchestrate enterprise processes across multiple systems, refer to the [Maestro](https://docs.uipath.com/maestro/automation-cloud/latest/user-guide/introduction-to-maestro) user guide.

## 15 April 2025

### Solutions in Studio Web in Public Preview

We are thrilled to announce a new way of working in Studio Web that makes the entire automation development lifecycle more efficient and scalable in an unified developer experience.

Solutions in Studio Web enable you to design and deploy your automations as a unified whole, integrating multiple projects and their resources. With solutions in the unified build experience, you can now work in a single environment to automate every aspect of an end-to-end business process.

To start building your first comprehensive solution in Studio Web, refer to in the Studio Web guide.

### Create individual projects

With the introduction of solutions in Studio Web, individual RPA projects can now be created by selecting the **Workflow** option from the arrow next to the **Create New** button. To account for this change, we have included all the information about RPA projects under a new section in the Studio Web guide.

You can also create individual projects by selecting the **Create New** button and choosing the project type from the **Create new project** window.
