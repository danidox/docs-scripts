## Resources

* You cannot rename a resource.
* If you remove a resource from your solution, you also need to remove it manually from the activities that are using it.
* While we continue adding to the list of supported resources, the current list is:
  + **Actions**: task catalog.
  + **Integration Service**: connections.
  + **Orchestrator**: assets, queues, storage buckets, processes, action apps.
