To leverage the power of the UiPath Platform<sup>TM</sup>, developers need to work with multiple designer canvases and resource builders available in different areas of the platform. This is especially visible during the design and deployment phases, when most automation components need to be created and configured in their platform context to become a working, cohesive mechanism.

The **unified developer experience** brings all the development tools together into one experience for the entire automation development lifecycle—from design to deployment. This means that you can now work in a single setting to automate all the areas of a business process.

Studio Web acts as the central, integrated environment to create everything you need for your automation solution, including the user interface layer, business logic layer, and data layer. Defining each of these layers within the context of Studio Web ensures a consistent development experience.

A **solution** groups everything needed to automate an end-to-end business process, offering you the capabilities to develop automation projects together with their dependencies and deploy them across tenants as a whole.

Before solutions, Studio Web managed only individual projects (RPA automations) and their respective published NuGet (NUPKG) packages. Every other platform component (assets, queues, storage buckets, etc.) had to be configured or created when performing the deployment.

With the unified developer experience in Studio Web, everything is integrated—making automation development more efficient and scalable:

* During design time, solutions optimize development by minimizing the need to leave the IDE (Studio Web) and by sharing the resources needed to automate the entire business process.
* During deployment, solutions reduce the manual effort and the potential for human errors when handling platform components by incorporating all the required dependencies into a single package for automatic deployment.

Solutions are also available in Studio Desktop, allowing you to take advantage of more development tools to automate complex business scenarios. In Studio Desktop, you can add and edit RPA projects with the Windows compatibility. To learn more, refer to [Solutions](https://docs.uipath.com/studio/standalone/latest/user-guide/solutions) in the Studio user guide.
:::note
Solutions are not supported in Studio LTS versions prior to 2025.10. Upgrade to a newer Studio LTS version or use [Studio STS](https://docs.uipath.com/studio/standalone/latest/release-notes/december-2024) to take advantage of solutions.
:::
