As an **App Test Developer**, you can leverage the power of Test Cloud to create and manage tests for both your applications and automations directly in **Studio Web**. These tests are integrated with Test Manager, allowing you to benefit from the full testing lifecycle: from design and debugging to execution and traceability.

**Tests** are a dedicated project type in Studio Web. They are created through a unidirectional integration with Test Managertests can only be initiated and linked from Test Manager, not the other way around.

Within a **Studio Web Tests** project, you can:

* Design test cases using supported activity packages such as Testing.Activities or UIAutomation.Activities.
* Debug your test cases directly in the browser.
* Collaborate with other users on the same Test projects, but not at the same time.
* Publish tests to Orchestrator for execution and test management.
