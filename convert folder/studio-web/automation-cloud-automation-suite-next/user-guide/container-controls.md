Use container controls to define the layout of an app and to position input controls within the app. There are four types of container controls:

* [Container Layout](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/container-layout#container-layout)
* [Container](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/container#container)
* [Page Container](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/page-container#page-container)
* [Page](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/page#page)
