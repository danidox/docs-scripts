Organizations can control Studio Web functionalities by enforcing governance policies from Automation Ops. For example, you can enable or disable certain features, control which activities and packages are available in Studio Web, or whether preview activities and packages can be used in Studio Web projects.

Find out more about [configuring](https://docs.uipath.com/automation-ops/automation-suite/2024.10/USER-GUIDE/configure-a-governance-policy) and [deploying policies](https://docs.uipath.com/automation-ops/automation-suite/2024.10/USER-GUIDE/deploy-governance-policies-top), and the settings available in [Studio Web policies](https://docs.uipath.com/automation-ops/automation-suite/2024.10/USER-GUIDE/configure-a-governance-policy).
