Use the [Autopilot chat](https://docs.uipath.com/autopilot/other/latest/overview/about-autopilot-chat) to create and edit API workflows with simple, natural-language instructions. Instead of configuring complex workflows manually, describe your desired outcomes, and Autopilot translates your input into structured automation steps.

Autopilot then guides you through workflow creation, suggests activities and code snippets, helps manage API connections, and handles testing and validation. It also offers real-time error handling by detecting potential issues and guiding you toward solutions.

Autopilot is available for every expression and has context over your workflow and object structures. Use it to your advantage to generate JavaScript code and expressions that can either be used as such or altered to your needs.
