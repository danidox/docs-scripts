To export an app project:

1. [Go to the Automations page in Studio Web](https://studio.uipath.com/projects).
2. From the **Automations** page, select **See more** &gt; **Download** on the right side of the project.
3. The project is downloaded as a UIP file which can be imported in Studio Web.

To import an app project:

1. From the **Automations** page, select the arrow next to **New project** on the upper-right side of the page.
2. Select the **Import project** option.
3. In the import dialog, either select **Choose** to locate and open a UIP file from your machine, or drag a UIP file from your machine's file explorer and drop it in the dialog.
4. The imported project is added at the top of your projects list.
