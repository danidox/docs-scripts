The **Automation details** page contains detailed information about each project listed in the **Automations** page. To reach the **Automation details** page, simply select the ![](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/440701) icon on the right of a project.

Dedicated buttons on the top of the page allow you to edit and share the project. Select ![docs image](https://documentationpicturerepo.blob.core.windows.net/migrated/More_VT.png) **See more** to access more options for your project:

* **Project settings** - Allows you to change the name and the description of the project. The window is resizable.
* **Change history** - Opens the change history window, where you can see details about each version of the project listed in chronological order. Selecting a version opens it in Studio Web in read-only mode.
* **Duplicate** - Creates a copy of the selected project.
* **Open in Studio Desktop** - [Opens the project in Studio Desktop](https://docs.uipath.com/studio-web/automation-suite/latest/everyone-user-guide/opening-a-project-in-studio-desktop). If Studio is not installed on your machine, a dialog is displayed giving you the option to download and install it.
* **Download** - Downloads the selected project as a UIP file which can be imported in Studio Web.
* **Delete** - Removes the selected project.
  :::note
  Deleting a project cannot be undone.
  :::

On the right side of the **Automation details** page, you can find other relevant project information:

* **Collaborators** - Who collaborated on the project.
* **Connections used** - What Integration Service connections are used in the project. Each connection contains the following information: an icon representing the connector, the connection name, the connection status, and whether a connection is personal or from a shared folder. The connection status is highlighted by the following icons:
  + The **Connected** ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/444289) icon if the connection is valid.
  + The **Disconnected** ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/444293) icon if there are issues with the connection. Select ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/444302) &gt; **Fix connection** on the right side of a disconnected connection to fix it. Please note that no status is available for browser connections. Select ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/444302) &gt; **Manage all connection** to handle your connections in the Integration Service
* **Project Details** - See the project's published status and when it was created, last modified, and last deployed.
  ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627900/444322)
