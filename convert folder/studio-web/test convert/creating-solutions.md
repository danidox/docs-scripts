To create a solution in Studio Web:

1. Go to [studio.uipath.com](http://studio.uipath.com/).
2. On the upper-right side of the page, select **Create New**.
3. In the new window, choose either **Agentic Process**, **Agent**, **RPA Workflow**, **API Workflow**, or **App**. Alternatively, select the arrow next to the **Create New** button and choose the type of solution you want to create. The new solution opens.
