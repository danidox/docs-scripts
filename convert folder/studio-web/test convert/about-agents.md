Studio Web enables you to build, test, and deploy UiPath® [Agents](https://docs.uipath.com/agents/automation-cloud/latest/user-guide/about-agents).

Agents are advanced automation components that can operate in dynamic environments. They use technologies such as large language models (LLMs), machine learning, and traditional enterprise automation to perform tasks that require judgment, flexibility, and contextual awareness.

You can create an agent project inside a solution. The Agents designer canvas in Studio Web provides a low-code interface for configuring all aspects of an agent. Here you can:

* Design and test agents, both autonomous and conversational
* Configure agent components
* Integrate agents into workflows
* Deploy agents as standalone processes
