To see the activity categories available in Studio Web, check out [Activities overview](https://docs.uipath.com/overview/other/latest/overview/activities-overview#activity-packages-in-studio-web) in the Overview guide. Click a category if you want to find out more about the activities it contains. The activities are also available in Studio and StudioX, and there can be differences in how they are configured depending on the project compatibility. Studio Web projects use the cross-platform compatibility.

To jump to the documentation of an activity you added to a project, select **Actions** ![](https://documentationpicturerepo.blob.core.windows.net/migrated/More_VT.png) &gt; **Help** for that activity in the project designer.
