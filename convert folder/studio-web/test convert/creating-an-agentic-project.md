To create an agentic project:

1. [Go to the Workspace page in Studio Web](https://studio.uipath.com/projects).
2. On the upper-right side of the page, select **Create New.**
3. Select **Agentic Process**. A new solution opens, with the **Agentic Process** designer canvas displayed.

Alternatively, you can select the arrow next to the **Create New** button, and then choose **Agentic Process** from the list of available project types.

Each agentic project consists of one or multiple agentic processes.

From the ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627891/503577) **Project Explorer** button on the upper-left side of the designer, you can:

* Use the ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627891/503563) **Add** button to add a new agentic process to your project or import an existing agentic process. You can import .bpmn files created with any BPMN 2.0-compatible tools.
* Use the ![docs image](https://docs-dev.uipath.com/api/binary/studio-web/2/627891/503567) **Search** button to find specific project files.
* Manage your project files. Right-click the top-level project to add an agentic process, import a .bpmn file, or rename, debug, or debug step-by-step your entire project. Right-click an individual agentic process (denoted by its .bpmn suffix) to rename, delete, or download the process file.
