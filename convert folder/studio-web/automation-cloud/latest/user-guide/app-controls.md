Controls form the basic building blocks of Apps and help build visually rich applications. You can access controls from the **Toolbox.**

There are four control categories:

* [Input Controls](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/input-controls#input-controls) facilitate user input inside your app.
* [Display Controls](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/display-controls#display-controls) allow you to display content in your app.
* [Container Controls](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/container-controls#container-controls) define the layout of an app and allow you to position input controls within the app.
* [Icons](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/icons#icon-controls) are a set of commonly used icons which you can add to your app.
