App activities allow you interact with app elements, as well as pass data between your app interface and the workflows used in your app. The activities are part of the [**App Events** package](https://docs.uipath.com/activities/other/latest/workflow/app-events-activities).

The **App Events** package includes the following activities:

* [Open Apps Page](https://docs.uipath.com/activities/other/latest/workflow/open-apps-page)
* [Close Apps pop-over/bottom-sheet](https://docs.uipath.com/activities/other/latest/workflow/close-apps-pop-over-bottom-sheet)
* [Show Apps Message](https://docs.uipath.com/activities/other/latest/workflow/show-apps-message)
* [Show/Hide Apps Spinner](https://docs.uipath.com/activities/other/latest/workflow/show-hide-apps-spinner)
* [Open URL in Apps](https://docs.uipath.com/activities/other/latest/workflow/open-url-in-apps)
* [Submit Action](https://docs.uipath.com/activities/other/latest/workflow/submit-action)
