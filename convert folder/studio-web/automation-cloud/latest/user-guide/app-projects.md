Studio Web enables you to build, debug, and deploy [UiPath<sup>®</sup> Apps](https://docs.uipath.com/apps/automation-cloud/latest/user-guide/introduction) using advanced developer tools that are tailored to your automation needs. This unified developer experience allows you to create apps powered by inline automation, which are then executed by serverless robots.

Developing Apps in Studio Web lets you:

* Have uninterrupted interactions with your automations;
* Build faster in a single automation platform;
* Create apps and forms backed by APIs and powerful UI and AI automation;
* Debug your app projects during development in an intuitive, friendly environment;
* Benefit from solid runtime performance.

To create an app project within a solution in Studio Web, select **Create New** &gt; **App**.

To get started quickly with an already-functional app project, check out [these templates](https://docs.uipath.com/studio-web/automation-cloud/latest/everyone-user-guide/web-app-templates).
