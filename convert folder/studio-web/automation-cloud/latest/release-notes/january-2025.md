## 29 January 2025

### New Date selector

Activities that use the **Date** selector to set a date in compatible activity fields no longer require you to also select the time. Fields that require both date and time will continue to ask for both values.

![Date selector](/images/sw/release-notes-date-selector-515637.webp)

### New flow for tabular controls and entities in Apps in Studio

You can now connect tabular controls such as the **Table, Edit Grid, Custom List, Dropdown,** and **Multiselect Dropdown** to your Data Service entities, as part of automation workflows in Web App projects.

Use the **Create workflow** option in the **Data source** field to add [Data Service-based activities](https://docs.uipath.com/activities/other/latest/workflow/about-the-dataservice-activities-pack) to your web app within Studio workflows. This allows you to create, query, edit, or delete records from your Data Service entities as a part of your automations.

## 21 January 2025

### New Custom List control

Web App projects now support the **Custom List** control, a dynamic template-based control which allows you to add other controls within its layout.

Learn how to use this control [in the Apps documentation](https://docs.uipath.com/apps/automation-cloud/latest/user-guide/custom-list).
