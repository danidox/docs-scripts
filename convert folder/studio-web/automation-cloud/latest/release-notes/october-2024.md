## 22 October 2024

### Web app projects in Studio Web in Public Preview

We are excited to announce that you can now build, test, and publish web app projects in Studio Web! To learn how you can create powerful web apps in a unified developer experience, refer to.

![Web app projects in Studio Web](/images/sw/release-notes-web-app-projects-in-studio-web-486297.webp)

### More options to debug your projects

You can now debug your projects using either cloud robots or your local robot. Simply select which testing environment you want to use from the new drop-down menu next to the testing button. To learn more, refer to.

![Test environments](/images/sw/release-notes-test-environments-466461.webp)

### Improved project navigation

We enhanced project navigation in Studio Web with the introduction of breadcrumb navigation. This means that the upper-left side of the designer now displays the multi-level structure of a project, in addition to the workflow name.
:::note
Project names can no longer be edited from upper-left side of the designer. Projects can be renamed from the **Project settings** context menu option in the , from the , or by right-clicking the project in the .
:::

![Project navigation](/images/sw/release-notes-project-navigation-476371.webp)

### Other improvements

When extracting an activity as a new workflow, the name of the workflow in the **New Workflow** window is now automatically filled with the activity’s name (without spaces between words).
