## 21 February 2025

### Apps in Studio Web are now Generally Available (GA)

We are thrilled to announce that Apps in Studio Web are now GA. Apps, previously known as web apps, allow you to design, test, and deploy enterprise-grade automations and applications under a single authoring experience.

The major highlights of this release are:

* All of the existing features from VB Apps, combined with the power of Studio Web workflows.
* Seamless integration with Actions, media, and Data Service entities.
* App activities which allow you to interact with app elements, and pass data between your app interface and workflows.
* Faster execution facilitated by serverless robots.
* Ready-to-use templates demonstrating several use cases.

Refer to to start building your apps in Studio Web.

:::note
You cannot publish a new version of an app project created during the Public Preview (a "Publish project failed" error appears). To prevent losing your work, either rename your project, or delete the previous package versions of the same project in Orchestrator before publishing. This issue does not affect new app projects.
:::
