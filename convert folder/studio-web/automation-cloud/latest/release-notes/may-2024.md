## 16 May 2024

### Extended trigger testing search results

When testing a trigger, the **Run output** window now shows more search results. If no items matching the search criteria are found, you can now expand the time interval of the search. Alternatively, you can select an item that does not match your search criteria. Additionally, matching items now have more readable names and are easier to identify. For more information, refer to.

![Extended trigger testing search results](/images/sw/release-notes-extended-trigger-testing-search-results-404888.webp)

### Introducing the Properties panel

To give you more control over activity properties configuration, you can access the new **Properties panel** from its corresponding icon on the upper-right side of the project page. The Properties panel allows you to switch between two views that either display all properties in the activity card or list the properties in the panel. For more information, refer to .

![The Properties panel](/images/sw/release-notes-the-properties-panel-404394.webp)

### Improved Run output filtering

You can now filter the messages in the **Run output** window by all severity levels (**Trace**, **Info**, **Warning**, **Error**, **Fatal**). The design of the severity level icons has also been updated.

![Run output filtering](/images/sw/release-notes-run-output-filtering-405104.webp)

### Improved search results in activity fields

To improve the experience of searching for matching items in activity fields, you can use the new **Load more** button to see more search results, even if your search does not return any results.
