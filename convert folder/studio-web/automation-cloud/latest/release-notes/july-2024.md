## 31 July 2024

### Improved activity searching

When searching for an activity, search results now prioritize activities from categories already added to your project.

### See the versions of a published project

You now have access to all the versions of a published project from the new **Change history** panel. To learn more about the panel, refer to .

### Simplified template configuration

When creating a project from a template, you now only need to customize connections with your data. Additional configuration options are available by enabling **Advanced configuration**.

![Additional configuration options](/images/sw/release-notes-additional-configuration-options-447468.webp)

## 1 July 2024

### Unleash your productivity with Autopilot™

Autopilot™ offers a set of AI-powered capabilities designed to help you create and test automations faster and easier. To see what Autopilot offers across the UiPath® Business Automation Platform, check out the [Autopilot guide](https://docs-staging.uipath.com/autopilot/other/latest/user-guide/autopilot-about).

Using Autopilot capabilities, you can utilize natural language to generate complex expressions. AI-generated expressions enable you to describe the desired outcome for a particular activity input, while an AI-based model will create a valid expression based on the request. For more information, refer to.

You can also use natural language prompts to describe to Autopilot the structure and outcome of a workflow. You can start building a workflow from scratch or from one of the predefined prompt examples. Based on your instructions, Autopilot identifies relevant activities and adds them in a logical sequence, providing a visual preview of the proposed workflow. You can also generate the next steps in your automation by typing in your instructions as annotations inside sequences. For more information, see and.

With its summarization capabilities, Autopilot can also describe the inner workings of activities and sequences by creating meaningful names that take into account the elements that define the activity or sequence. To learn more, refer to.

  ![Create a project with Autopilot](/images/sw/release-notes-create-a-project-with-autopilot-436155.webp)
