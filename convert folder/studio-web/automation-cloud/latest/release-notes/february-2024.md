## 9 February 2024

### Improvements

* To make it easier to select the variable property you need, when you open the variable selector in an activity field, the variable of the previous activity that has an output is expanded by default.
* Switching between files in a project with multiple workflows is now faster.
* We improved the performance of the condition builder.

## 7 February 2024

### Variables and arguments automatically added to Watches

You can now watch the values of variables and arguments when without manually adding them to the watches. For variables with multiple properties, you can expand the entries in the Watches panel to view the values of all available properties.
