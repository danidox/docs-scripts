## 27 June 2025

### Introducing Unified Pricing

We are excited to announce Unified Pricing, our innovative licensing paradigm that brings new licensing plans, user licenses, and a consolidated consumption unit. We are also excited to let you know Studio Web is now available through Unified Pricing.

For extensive release notes, refer to the [Automation Cloud release notes guide](https://docs.uipath.com/automation-cloud/automation-cloud/latest/release-notes/may-2025).

For detailed Studio Web licensing information, refer to the [Studio user guide](https://docs.uipath.com/studio/standalone/latest/user-guide/about-licensing-unified-pricing).

For overview licensing information, refer to the [Automation Cloud admin guide](https://docs.uipath.com/automation-cloud/automation-cloud/latest/admin-guide/about-licensing) and the [Overview guide](https://docs.uipath.com/overview/other/latest/overview/licensing-models).
