## 28 March 2025

### Agentic processes in Public Preview

We are excited to introduce agentic processes, a new type of project in Studio Web that enables you to model and implement complex business processes using Business Process Model and Notation (BPMN) concepts.

With agentic processes, you can orchestrate long running processes with complex flow control logic by integrating automated, people, agent, and decision-based tasks while leveraging UI, API, and AI-powered capabilities.

For more information on how to work with agentic processes, refer to in the Studio Web user guide.

For more information on how to orchestrate enterprise processes, refer to the [Maestro user guide](https://docs.uipath.com/maestro/automation-cloud/latest/user-guide/introduction-to-maestro).

It's now easier to choose the type of project when creating a new project. Select the **Create New** button and choose a project type from the new project selection window, or select the arrow next to the button to open the list of available project types.
