## 12 June 2024

### Studio Web browser extensions updates

In response to [Google’s timeline for Chrome Manifest V2 (MV2) deprecation](https://developer.chrome.com/docs/extensions/develop/migrate/mv2-deprecation-timeline), starting from June 2024, existing Studio Web extensions for Chrome and Edge have been updated to be fully compatible with Manifest V3, while retaining the same extension IDs for both browsers ([Chrome](https://chrome.google.com/webstore/detail/uipath-extension-for-stud/conkfbpnllelocpogdmbilgmnkabjfmf) and [Edge](https://microsoftedge.microsoft.com/addons/detail/uipath-extension-for-stud/hmpkmollbbcoopahpbplbdaapacncbbc)).
