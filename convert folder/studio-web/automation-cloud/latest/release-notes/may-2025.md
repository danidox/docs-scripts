## 14 May 2025

### UiPath Agents in solutions are now Generally Available (GA)

You can now build agents as part of a solution in Studio Web. Building agents in Studio Web streamlines the lifecycle management of agents, their tools, and your agentic automations.

Please note that you currently cannot add multiple agents or other project types to a solution that contains an agent.

To learn how you can create, configure, publish, and run agents, refer to [Agents](https://docs.uipath.com/studio-web/automation-cloud/latest/user-guide/about-agents) in the Studio Web user guide.

To learn more about the powerful capabilities of agents, refer to the [Agents user guide](https://docs.uipath.com/agents/automation-cloud/latest/user-guide/about-agents).

### Improvements

When working in a solution, you can now delete the workflow file that is currently open or the folder that contains the active workflow file.
