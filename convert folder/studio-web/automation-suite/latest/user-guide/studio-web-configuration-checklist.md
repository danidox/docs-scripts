## **Create and debug a Studio Web project**

Follow the steps below to make sure Studio Web works as expected in Automation Suite:

1. Create a new Studio Web project.
2. Add a [Write Line](https://docs.uipath.com/activities/other/latest/workflow/write-line) activity to your workflow.
3. Select the **Test step-by-step** debugging option.
4. Wait for the details about the run to appear in the **Run output** panel.
   :::note
   Most activities are not immediately available. Please allow approximately 40 to 60 minutes for all activities categories to load from the official UiPath feed.
   :::
