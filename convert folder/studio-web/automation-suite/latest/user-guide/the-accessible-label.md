## About the Accessible label

Screen readers use the value of the **Accessible** label as an accessible descriptor.

The **Accessible** label is an expressionable VB property. You can bind it to other properties in your app. For example, if a **Button** control on the **MainPage** of your app uses the **Accessible** label, you can reference it using the following expression: `MainPage.Button.AccessibleLabel`.

### Using the Accessible label

To add accessible descriptions using the **Accessible** labels in your app:

1. Open your app.
2. Select a **Button** or **Custom HTML** control. The **Properties** panel opens.
3. Add accessible text to the **Accessible** label property.
