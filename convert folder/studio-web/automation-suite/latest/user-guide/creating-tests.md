To create a **Tests** project in Studio Web, you must initiate the process from Test Manager by selecting a test case and choosing **Automate in Studio Web**.

This action triggers the creation of a Studio Web **Tests** project and optionally a new test case within it. Once the project and test case are created, they become available in Studio Web for designing and debugging.
:::note
You cannot create a **Tests** project directly from Studio Web. This flow is only available via Test Manager.
:::
For detailed steps on creating a Tests project in Studio Web, visit [Automating Test Manager test cases in Studio Web](https://docs.uipath.com/test-manager/automation-suite/latest/user-guide/automating-test-cases-in-studio-web).
