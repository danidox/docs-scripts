These mobile-friendly app project templates are designed to showcase the capabilities of UiPath<sup>®</sup> Apps powered by Studio Web workflows.

Each template includes example company branding to ensure a professional appearance, logic to handle any errors that might arise, and activities which use connections that you can easily configure or replace. These templates enable you to have a fully functioning app powered by Studio Web ready for deployment in minutes.

## File Uploader App using Google Drive

This app uploads files to Google Drive using Studio automations.

**Integration Service connection:** [Google Drive activities](https://docs.uipath.com/activities/other/latest/productivity/drive-activities)

[**Use template**](https://cloud.uipath.com/portal_/cloudrpa?redirectPath=studio_/templates/cb19163c-6fb6-44bd-9c9a-038068f5e250)
